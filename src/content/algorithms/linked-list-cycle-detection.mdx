## Why It Surfaces

This is the clean pointer interview question: detect a loop with constant extra space. It often starts with a yes-or-no check, then extends into finding the node where the cycle begins.

## JavaScript

```js
function hasCycle(head) {
  let slow = head;
  let fast = head;

  while (fast !== null && fast.next !== null) {
    slow = slow.next;
    fast = fast.next.next;

    if (slow === fast) {
      return true;
    }
  }

  return false;
}
```

## TypeScript

```ts
type Node = {
  value: number;
  next: Node | null;
};

export function hasCycle(head: Node | null): boolean {
  let slow = head;
  let fast = head;

  while (fast !== null && fast.next !== null) {
    slow = slow?.next ?? null;
    fast = fast.next.next;

    if (slow !== null && slow === fast) {
      return true;
    }
  }

  return false;
}
```

## Interview Notes

- Explain why two pointers must eventually meet inside a cycle.
- Mention that resetting one pointer to the head helps locate the cycle start.
- Contrast with a hash-set solution if the interviewer asks for a simpler but less space-efficient approach.
