## Why It Surfaces

Matrix exponentiation appears when an interviewer wants a recurrence faster than linear time. Once a recurrence is encoded as a transition matrix, repeated application becomes a power problem instead of a step-by-step simulation.

## Worked Example

If a recurrence advances by multiplying a state vector with matrix `M`, then applying that transition `n` times means computing `M^n`. Exponentiation by squaring reduces that from `n` matrix multiplications to `log n` levels.

<Callout title="Base case" tone="tip">
  The zero power of any square matrix is the identity matrix of the same dimension, not the original matrix.
</Callout>

<ExamplePair>

```js
function multiplyMatrices(left, right) {
  const size = left.length;
  const result = Array.from({ length: size }, () => new Array(size).fill(0));

  for (let row = 0; row < size; row += 1) {
    for (let col = 0; col < size; col += 1) {
      for (let index = 0; index < size; index += 1) {
        result[row][col] += left[row][index] * right[index][col];
      }
    }
  }

  return result;
}

function identityMatrix(size) {
  return Array.from({ length: size }, (_, row) =>
    Array.from({ length: size }, (_, col) => (row === col ? 1 : 0)),
  );
}

function matrixExponentiation(matrix, power) {
  if (power === 0) {
    return identityMatrix(matrix.length);
  }

  if (power % 2 === 1) {
    return multiplyMatrices(matrixExponentiation(matrix, power - 1), matrix);
  }

  const half = matrixExponentiation(matrix, Math.floor(power / 2));
  return multiplyMatrices(half, half);
}
```

  return result;

  );
}

export function matrixExponentiation(matrix: number[][], power: number): number[][] {

    return identityMatrix(matrix.length);

    return multiplyMatrices(matrixExponentiation(matrix, power - 1), matrix);

  const half = matrixExponentiation(matrix, Math.floor(power / 2));

}

## Follow-up Questions

## Common Pitfalls

- Returning the original matrix instead of the identity matrix for power zero.
- Confusing element-wise multiplication with matrix multiplication.
- Missing the logarithmic win and multiplying the matrix linearly `p` times.
