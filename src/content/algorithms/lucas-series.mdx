## Why It Surfaces

Lucas numbers are useful because they force you to separate the recurrence from the initial conditions. If you only memorize Fibonacci, you usually get this wrong on the first try.

## Worked Example

The series starts `2, 1`. That gives `L(2) = 3`, `L(3) = 4`, and `L(4) = 7`, since every term is still the sum of the previous two.

## Reference Implementations

<ExamplePair>

```js
function lucasSeries(index) {
  if (!Number.isInteger(index) || index < 0) {
    throw new TypeError('Index must be a non-negative integer');
  }

  let previous = 2;
  let current = 1;

  for (let step = 0; step < index; step += 1) {
    const next = previous + current;
    previous = current;
    current = next;
  }

  return previous;
}
```

## Common Pitfalls

- Reusing Fibonacci base cases by mistake.
- Returning the wrong variable after the loop because of off-by-one indexing.
- Falling back to naive recursion and recreating the exponential call tree.
