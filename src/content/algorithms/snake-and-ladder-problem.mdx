## Why It Surfaces

Snake and Ladder is a clean example of BFS on an implicit graph. The board is not given as adjacency lists, but each square still has outgoing edges to the next six dice outcomes after accounting for any snake or ladder jump.

## Worked Example

If square `3` contains a ladder to `14`, then from square `1` a roll of `2` really reaches `14` in one move, not `3`. BFS handles that naturally by enqueuing the redirected destination with one more dice throw.

## Reference Implementations

<ExamplePair>

```js
function snakeAndLadderProblem(board) {
  const target = board.length - 1;
  const visited = new Array(board.length).fill(false);
  const queue = [[0, 0]];
  visited[0] = true;

  for (let head = 0; head < queue.length; head += 1) {
    const [cell, throwsUsed] = queue[head];
    if (cell === target) {
      return throwsUsed;
    }

    for (let move = 1; move <= 6 && cell + move < board.length; move += 1) {
      let next = cell + move;
      if (board[next] !== -1) {
        next = board[next];
      }

      if (!visited[next]) {
        visited[next] = true;
        queue.push([next, throwsUsed + 1]);
      }
    }
  }

  return -1;
}
```

## Common Pitfalls

- Marking the pre-redirect square visited instead of the actual destination square.
- Counting a snake or ladder jump as an extra move when it is part of the same dice throw.
- Using DFS or brute force when the prompt asks for a minimum number of throws.
