## Why It Surfaces

The formula is a short composition of two standard geometry ideas: area of a sector and area of the isosceles triangle formed by the chord endpoints and the center. The prompt is useful because it checks whether you recognize that a segment is a sector with the triangle carved out.

## Worked Example

For `radius = 10` and `angle = 90`, the minor segment area is about `28.5398` and the major segment area is about `285.6195`.

## Reference Implementations

<ExamplePair>

```js
function programToFindAreaOfACircularSegment(radius, angleDegrees) {
  if (
    !Number.isFinite(radius)
    || !Number.isFinite(angleDegrees)
    || radius < 0
    || angleDegrees < 0
    || angleDegrees > 180
  ) {
    throw new RangeError("Radius must be non-negative and angle must be between 0 and 180 degrees");
  }

  const angleRadians = (angleDegrees * Math.PI) / 180;
  const sectorArea = Math.PI * radius * radius * (angleDegrees / 360);
  const triangleArea = 0.5 * radius * radius * Math.sin(angleRadians);
  const minorSegmentArea = sectorArea - triangleArea;
  const circleArea = Math.PI * radius * radius;

  return {
    minorSegmentArea,
    majorSegmentArea: circleArea - minorSegmentArea,
  };
}
```

## Common Pitfalls

- Returning the sector area directly without subtracting the triangle.
- Passing degrees into `Math.sin` without converting to radians.
- Accepting angles larger than `180` even though the prompt describes the minor segment.
