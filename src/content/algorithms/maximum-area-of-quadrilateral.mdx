## Why It Surfaces

This prompt is more interesting than a direct area formula because the quadrilateral itself is not fixed. The useful fact is that among all quadrilaterals with the same four sides, the cyclic one maximizes the area, so Bretschneider's formula collapses to Brahmagupta's formula.

## Worked Example

For sides `1`, `2`, `1`, and `2`, the semiperimeter is `3`, so the maximum area is `sqrt((3 - 1) * (3 - 2) * (3 - 1) * (3 - 2)) = 2`.

<ExamplePair>

```js
function maximumAreaOfQuadrilateral(a, b, c, d) {
  const sides = [a, b, c, d];
  if (!sides.every(Number.isFinite) || sides.some((side) => side <= 0)) {
    throw new RangeError("Side lengths must be positive finite numbers");
  }

  const perimeter = a + b + c + d;
  const longestSide = Math.max(...sides);
  if (longestSide >= perimeter - longestSide) {
    throw new RangeError("The given lengths cannot form a non-degenerate quadrilateral");
  }

  const semiperimeter = perimeter / 2;
  return Math.sqrt(
    (semiperimeter - a)
    * (semiperimeter - b)
    * (semiperimeter - c)
    * (semiperimeter - d),
  );
}
```

## Common Pitfalls

- Using Brahmagupta's formula without noting that it gives the maximum-area case here.
- Forgetting to validate that the four side lengths can form a quadrilateral.
- Mixing the semiperimeter formula up with the triangle-only Heron case.
