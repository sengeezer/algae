## Why It Surfaces

Koch Snowflake is a good recursion problem because the rule is local and mechanical: every segment becomes four smaller segments. The interesting part is not traversal, but building the new geometry consistently from vector arithmetic.

## Worked Example

Starting with the segment from `(0, 0)` to `(3, 0)`, the first refinement inserts points at `(1, 0)` and `(2, 0)` and replaces the middle third with the apex of an equilateral triangle at approximately `(1.5, 0.866)`. One segment becomes four.

function iterateKoch(points, steps) {
  if (!Number.isInteger(steps) || steps < 0) {
    throw new RangeError('Steps must be a non-negative integer');
  }

  let current = points.map((point) => ({ ...point }));

  for (let step = 0; step < steps; step += 1) {
    current = kochStep(current);
  }

  return current;
}

function kochStep(points) {
  const next = [];

  for (let index = 0; index < points.length - 1; index += 1) {
    const start = points[index];
    const end = points[index + 1];
    const delta = {
      x: (end.x - start.x) / 3,
      y: (end.y - start.y) / 3,
    };
    const firstThird = {
      x: start.x + delta.x,
      y: start.y + delta.y,
    };
    const rotated = rotate60(delta);
    const peak = {
      x: firstThird.x + rotated.x,
      y: firstThird.y + rotated.y,
    };
    const secondThird = {
      x: start.x + 2 * delta.x,
      y: start.y + 2 * delta.y,
    };

    next.push(start, firstThird, peak, secondThird);
  }

  next.push(points[points.length - 1]);
  return next;
}

function rotate60(vector) {
  const radians = Math.PI / 3;
  return {
    x: vector.x * Math.cos(radians) - vector.y * Math.sin(radians),
    y: vector.x * Math.sin(radians) + vector.y * Math.cos(radians),
  };
}

## Follow-up Questions

- Start from an equilateral triangle and generate the full closed snowflake boundary.
- Explain why the segment count quadruples at each step.
- Render the points to a canvas or SVG path.

## Common Pitfalls

- Forgetting that the input polyline should include adjacent endpoint pairs in order.
- Rotating the wrong third of the segment or using the wrong sign for the angle.
- Using too many iterations without accounting for exponential growth in the point list.
