## Why It Surfaces

Month-length lookup is a small calendar routine that still exposes real edge cases. The interviewer usually wants to see whether you remember that leap years are not just "divisible by 4."

## Worked Example

`(2, 2024)` returns `29`, while `(2, 2023)` returns `28`. All other months fall into the fixed `30`-day or `31`-day groups.

## Reference Implementations

<ExamplePair>

```js
const THIRTY_ONE_DAY_MONTHS = [1, 3, 5, 7, 8, 10, 12];
const THIRTY_DAY_MONTHS = [4, 6, 9, 11];

function isLeapYear(year) {
  if (!Number.isInteger(year) || year <= 0) {
    throw new RangeError('Year must be a positive integer');
  }

  return year % 400 === 0 || (year % 4 === 0 && year % 100 !== 0);
}

function getMonthDays(month, year) {
  if (!Number.isInteger(month) || month < 1 || month > 12) {
    throw new RangeError('Month must be an integer from 1 to 12');
  }

  if (THIRTY_ONE_DAY_MONTHS.includes(month)) {
    return 31;
  }

  if (THIRTY_DAY_MONTHS.includes(month)) {
    return 30;
  }

  return isLeapYear(year) ? 29 : 28;
}
```

```ts
const THIRTY_ONE_DAY_MONTHS = [1, 3, 5, 7, 8, 10, 12] as const;
const THIRTY_DAY_MONTHS = [4, 6, 9, 11] as const;

function isLeapYear(year: number): boolean {
  if (!Number.isInteger(year) || year <= 0) {
    throw new RangeError('Year must be a positive integer');
  }

  return year % 400 === 0 || (year % 4 === 0 && year % 100 !== 0);
}

export function getMonthDays(month: number, year: number): number {
  if (!Number.isInteger(month) || month < 1 || month > 12) {
    throw new RangeError('Month must be an integer from 1 to 12');
  }

  if (THIRTY_ONE_DAY_MONTHS.includes(month as (typeof THIRTY_ONE_DAY_MONTHS)[number])) {
    return 31;
  }

  if (THIRTY_DAY_MONTHS.includes(month as (typeof THIRTY_DAY_MONTHS)[number])) {
    return 30;
  }

  return isLeapYear(year) ? 29 : 28;
}
```

## Follow-up Questions

- Use this helper inside a `parseDate` validator.
- Explain why years like `1900` are not leap years but `2000` is.
- Return the month name and day count together for UI display code.

## Common Pitfalls

- Treating every year divisible by `4` as a leap year.
- Mixing zero-based month indexes with human month numbers.
- Accepting month `0` or `13` without rejecting the input.
