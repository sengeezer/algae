## Why It Surfaces

Row-echelon questions test whether you can manage matrix-wide state carefully: choose pivots, swap rows when needed, and keep the elimination numerically sane.

## Worked Example

Starting from a rectangular or square matrix, choose the first nonzero pivot in the current column, move it into place, scale the row so the pivot becomes `1`, and subtract multiples of that row from the rows below until everything under the pivot is `0`.

```js
function validateMatrix(matrix) {
  if (!Array.isArray(matrix) || matrix.length === 0 || !matrix.every((row) => Array.isArray(row))) {
    throw new TypeError('Expected a non-empty 2D matrix');
  }

  const width = matrix[0].length;
  if (width === 0 || !matrix.every((row) => row.length === width && row.every((value) => Number.isFinite(value)))) {
    throw new TypeError('Matrix rows must have equal length and contain only finite numbers');
  }
}

function rowEchelon(matrix) {
  validateMatrix(matrix);

  const result = matrix.map((row) => row.slice());
  const rows = result.length;
  const cols = result[0].length;
  const tolerance = 1e-9;

  let pivotRow = 0;
  for (let pivotCol = 0; pivotRow < rows && pivotCol < cols; pivotCol += 1) {
    let bestRow = pivotRow;
    while (bestRow < rows && Math.abs(result[bestRow][pivotCol]) <= tolerance) {
      bestRow += 1;
    }

    if (bestRow === rows) {
      continue;
    }

    [result[pivotRow], result[bestRow]] = [result[bestRow], result[pivotRow]];

    const pivotValue = result[pivotRow][pivotCol];
    for (let col = pivotCol; col < cols; col += 1) {
      result[pivotRow][col] /= pivotValue;
    }

    for (let row = pivotRow + 1; row < rows; row += 1) {
      const factor = result[row][pivotCol];
      if (Math.abs(factor) <= tolerance) {
        continue;
      }

      for (let col = pivotCol; col < cols; col += 1) {
        result[row][col] -= factor * result[pivotRow][col];
        if (Math.abs(result[row][col]) <= tolerance) {
          result[row][col] = 0;
        }
      }
    }

    pivotRow += 1;
  }

  return result;
}
```

## Follow-up Questions

- Use the elimination process to solve a linear system.
- Explain why row swaps matter when the current pivot candidate is zero.

## Common Pitfalls

- Assuming the matrix is square when the routine only needs a rectangular 2D array.
- Dividing by a near-zero pivot instead of swapping rows first.
- Mutating the input matrix when the caller expects a pure transformation.
