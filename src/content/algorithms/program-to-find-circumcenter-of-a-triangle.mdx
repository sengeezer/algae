## Why It Surfaces

This is the triangle-center version of the line-intersection problems from the previous slice. The geometric fact is simple: the circumcenter lies at the intersection of the perpendicular bisectors of the sides, so two bisectors are enough to recover it.

## Worked Example

For `P = [6, 0]`, `Q = [0, 0]`, and `R = [0, 8]`, the circumcenter is `[3, 4]`.

## Reference Implementations

```js
function programToFindCircumcenterOfATriangle(p, q, r) {
  const denominator = 2 * (
    p[0] * (q[1] - r[1])
    + q[0] * (r[1] - p[1])
    + r[0] * (p[1] - q[1])
  );

  if (denominator === 0) {
    return null;
  }

  const pSquared = p[0] * p[0] + p[1] * p[1];
  const qSquared = q[0] * q[0] + q[1] * q[1];
  const rSquared = r[0] * r[0] + r[1] * r[1];

  const x = (
    pSquared * (q[1] - r[1])
    + qSquared * (r[1] - p[1])
    + rSquared * (p[1] - q[1])
  ) / denominator;
  const y = (
    pSquared * (r[0] - q[0])
    + qSquared * (p[0] - r[0])
    + rSquared * (q[0] - p[0])
  ) / denominator;

  return [x, y];
}
```
