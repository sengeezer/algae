## Why It Surfaces

Automorphic numbers are small but interesting because they convert a verbal digit-pattern rule into a concrete suffix comparison based on modulo arithmetic.

## Worked Example

`25` is automorphic because $25^2 = 625$, and the trailing digits `25` match the original number. `7` is not, because $7^2 = 49$ does not end in `7`.

## Reference Implementations

<ExamplePair>

```js
function automorphicNumber(value) {
  if (!Number.isInteger(value) || value < 0) {
    return false;
  }

  let modulus = 10;
  let remaining = value;

  while (remaining >= 10) {
    modulus *= 10;
    remaining = Math.floor(remaining / 10);
  }

  return (value * value) % modulus === value;
}
```

```ts

  if (!Number.isInteger(value) || value < 0) {

}
```

## Follow-up Questions

- Compare the arithmetic modulus approach with converting numbers to strings and checking suffixes.
- Extend the idea to other bases besides decimal.
- Explain why `0`, `1`, `5`, and `6` are quick trivial examples.

## Common Pitfalls

- Miscounting how many decimal digits the suffix needs.
- Treating negative inputs as valid under this decimal-suffix definition.
- Forgetting that the modulus must be `10^d`, not just the digit count `d`.
