## Why It Surfaces

Catalan Numbers usually show up disguised as a counting problem. The interviewer may ask about balanced parentheses, valid BST shapes, or polygon triangulations, but the core recurrence is the same: choose a split point, then multiply the number of valid left and right structures.

## Worked Example

To compute `C(3)`, choose each possible root position. The first root contributes `C(0) * C(2)`, the second contributes `C(1) * C(1)`, and the third contributes `C(2) * C(0)`. Summing those cases gives `5`.

<Callout title="Recognition cue" tone="note">
  If every valid structure can be assembled from an independent left part and right part around a root, Catalan-style counting should be on your shortlist.
</Callout>

## Reference Implementations

<ExamplePair>

```js
function catalanNumber(index) {
  const dp = new Array(index + 1).fill(0);
  dp[0] = 1;

  for (let total = 1; total <= index; total += 1) {
    for (let leftSize = 0; leftSize < total; leftSize += 1) {
      dp[total] += dp[leftSize] * dp[total - 1 - leftSize];
    }
  }

  return dp[index];
}
```

```ts
export function catalanNumber(index: number): number {

  dp[0] = 1;

    for (let leftSize = 0; leftSize < total; leftSize += 1) {
      dp[total] += dp[leftSize] * dp[total - 1 - leftSize];
    }
  }

  return dp[index];
}
```

## Common Pitfalls

- Adding left and right counts when they should be multiplied.
- Missing the base case `C(0) = 1`.
- Failing to recognize the same recurrence under a different story problem.
