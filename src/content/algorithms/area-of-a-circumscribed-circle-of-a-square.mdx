## Why It Surfaces

This is a short formula prompt, but it checks whether you can connect the square diagonal to the circumcircle radius. Once that relation is recognized, the area collapses to a single expression instead of several separate geometry steps.

## Worked Example

For side length `6`, the area is `pi * 6^2 / 2 = 18pi`, which is about `56.5487`.

## Reference Implementations

<ExamplePair>

```js
function areaOfACircumscribedCircleOfASquare(side) {
  if (!Number.isFinite(side) || side < 0) {
    throw new RangeError("Side length must be a non-negative number");
  }

  return (Math.PI * side * side) / 2;
}
```

```ts
export function areaOfACircumscribedCircleOfASquare(side: number): number {
  if (!Number.isFinite(side) || side < 0) {
    throw new RangeError('Side length must be a non-negative number');
  }

  return (Math.PI * side * side) / 2;
}
```

## Common Pitfalls

- Using the side length directly as the radius.
- Confusing circumscribed and inscribed circles.
- Forgetting that the diagonal of a square is `side * sqrt(2)`.
