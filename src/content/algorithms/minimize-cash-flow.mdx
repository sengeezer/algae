## Why It Surfaces

The greedy insight is that the raw transaction edges do not matter once you know each person's net balance. After that reduction, every step should directly match the largest debtor with the largest creditor so that one side of the debt disappears immediately.

## Worked Example

For

```text
[
  [0, 1000, 2000],
  [0,    0, 5000],
  [0,    0,    0],
]
```

the net balances are `[-3000, -4000, 7000]`. Friend `0` should pay `3000` to friend `2`, and friend `1` should pay `4000` to friend `2`, so the minimized settlement matrix becomes:

```text
[
  [0, 0, 3000],
  [0, 0, 4000],
  [0, 0,    0],
]
```

<ExamplePair>

```js
function minimizeCashFlow(transactions) {
  const friendCount = transactions.length;

  if (friendCount === 0) {
    return [];
  }

  const balances = Array(friendCount).fill(0);
  const settlements = Array.from({ length: friendCount }, () =>
    Array(friendCount).fill(0),
  );

  for (let from = 0; from < friendCount; from += 1) {
    for (let to = 0; to < friendCount; to += 1) {
      balances[from] -= transactions[from][to];
      balances[to] += transactions[from][to];
    }
  }

  while (true) {
    let creditor = 0;
    let debtor = 0;

    for (let person = 1; person < friendCount; person += 1) {
      if (balances[person] > balances[creditor]) {
        creditor = person;
      }

      if (balances[person] < balances[debtor]) {
        debtor = person;
      }
    }

    if (balances[creditor] === 0 && balances[debtor] === 0) {
      break;
    }

    const amount = Math.min(balances[creditor], -balances[debtor]);
    settlements[debtor][creditor] += amount;
    balances[creditor] -= amount;
    balances[debtor] += amount;
  }

  return settlements;
}
```

## Follow-up Questions

- Return a compact list of non-zero transfers instead of a full settlement matrix.
- Replace the repeated max-debtor and max-creditor scan with heaps.
- Explain why settling net balances is enough even if the original graph has cycles.

## Common Pitfalls

- Trying to settle directly from the original matrix instead of first collapsing each person's balance.
- Forgetting that a greedy transfer should use the minimum of what the debtor owes and the creditor should receive.
- Treating zero-balance people as still active in later settlement rounds.
