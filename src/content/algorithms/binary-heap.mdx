## Why It Surfaces

Binary heaps show up whenever the problem needs the next highest-priority item over and over again. They preserve just enough order to keep the best candidate at the root without paying the cost of maintaining a fully sorted array.

<Callout title="Index rule" tone="note">
  In a zero-based heap array, the children of `index` live at `2 * index + 1` and `2 * index + 2`. That arithmetic is often what the interviewer is really testing.
</Callout>

## Worked Example

In a min heap holding `[2, 5, 4, 9, 7, 8]`, removing the root first swaps in the last element, then sifts it downward until the heap property is restored.

<ExamplePair>

```js
class MinHeap {
  constructor(values = []) {
    this.heap = [...values];

    for (let index = Math.floor(this.heap.length / 2) - 1; index >= 0; index -= 1) {
      this.siftDown(index);
    }
  }

  siftUp(index) {
    let child = index;

    while (child > 0) {
      const parent = Math.floor((child - 1) / 2);

      if (this.heap[parent] <= this.heap[child]) {
        break;
      }

      [this.heap[parent], this.heap[child]] = [this.heap[child], this.heap[parent]];
      child = parent;
    }
  }

  siftDown(index) {
    let parent = index;

    while (true) {
      let candidate = parent;
      const left = parent * 2 + 1;
      const right = parent * 2 + 2;

      if (left < this.heap.length && this.heap[left] < this.heap[candidate]) {
        candidate = left;
      }

      if (right < this.heap.length && this.heap[right] < this.heap[candidate]) {
        candidate = right;
      }

      if (candidate === parent) {
        return;
      }

      [this.heap[parent], this.heap[candidate]] = [this.heap[candidate], this.heap[parent]];
      parent = candidate;
    }
  }

  push(value) {
    this.heap.push(value);
    this.siftUp(this.heap.length - 1);
  }

  pop() {
    if (this.heap.length === 0) {
      return undefined;
    }

    const top = this.heap[0];
    const last = this.heap.pop();

    if (this.heap.length > 0 && last !== undefined) {
      this.heap[0] = last;
      this.siftDown(0);
    }

    return top;
  }

  peek() {
    return this.heap[0];
  }
}
```

      this.siftDown(index);
    }

  peek(): number | undefined {
    return this.heap[0];
  }

    this.siftUp(this.heap.length - 1);

## Common Pitfalls

- Mixing up parent and child index formulas.
- Forgetting to restore the heap after swapping the root with the last element.
- Assuming the array is globally sorted just because it is heap-ordered.
