## Why It Surfaces

CRC questions test whether you can reason about bit-level error detection rather than only about ordinary arithmetic. The key shift is that subtraction becomes XOR, because the division happens over binary coefficients modulo `2`.

## Worked Example

For data `100100` and key `1101`, modulo-2 division produces remainder `001`, so the transmitted codeword is `100100001`. Running the same division on `100100001` returns all zeros, so the received data is valid.

```js
function validateBinaryString(value, name) {
  if (typeof value !== 'string' || !/^[01]+$/.test(value)) {
    throw new TypeError(`${name} must be a non-empty binary string`);
  }
}

function xorFromSecondBit(left, right) {
  let result = '';

  for (let index = 1; index < right.length; index += 1) {
    result += left[index] === right[index] ? '0' : '1';
  }

  return result;
}

function mod2div(dividend, divisor) {
  let pick = divisor.length;
  let window = dividend.slice(0, pick);
  const zeroBlock = '0'.repeat(divisor.length);

  while (pick < dividend.length) {
    if (window[0] === '1') {
      window = xorFromSecondBit(divisor, window) + dividend[pick];
    } else {
      window = xorFromSecondBit(zeroBlock, window) + dividend[pick];
    }

    pick += 1;
  }

  return window[0] === '1'
    ? xorFromSecondBit(divisor, window)
    : xorFromSecondBit(zeroBlock, window);
}

function encodeCyclicRedundancyCheck(data, key) {
  validateBinaryString(data, 'data');
  validateBinaryString(key, 'key');

  if (!key.includes('1') || key.length < 2) {
    throw new RangeError('key must contain at least one 1-bit and have length >= 2');
  }

  const paddedData = data + '0'.repeat(key.length - 1);
  const remainder = mod2div(paddedData, key).padStart(key.length - 1, '0');

  return {
    remainder,
    encodedData: data + remainder,
  };
}

function verifyCyclicRedundancyCheck(codeword, key) {
  validateBinaryString(codeword, 'codeword');
  validateBinaryString(key, 'key');

  if (codeword.length < key.length) {
    throw new RangeError('codeword length must be at least the key length');
  }

  const remainder = mod2div(codeword, key).padStart(key.length - 1, '0');

  return {
    remainder,
    isValid: !remainder.includes('1'),
  };
}
```

## Follow-up Questions

- Accept a generator polynomial like `x^3 + x^2 + 1` and convert it to a binary key.
- Compare CRC with simpler parity-bit error detection.
- Stream the modulo-2 division one bit at a time instead of materializing full strings.

## Common Pitfalls

- Using ordinary subtraction instead of XOR during the long-division steps.
- Forgetting to append `key.length - 1` zeros before sender-side division.
- Treating a non-zero receiver remainder as valid data.
