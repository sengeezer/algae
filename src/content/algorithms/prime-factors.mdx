## Why It Surfaces

Prime factorization is one of the most reusable reductions in number theory. Once a value is broken into primes, divisor counts, totients, and many gcd-style arguments become direct formulas.

## Worked Example

For `84`, divide by `2` twice to get `21`, then divide by `3` once to get `7`. The prime factorization is `[2, 2, 3, 7]`.

## Reference Implementations

<ExamplePair>

```js
function primeFactors(value) {
  const factors = [];
  let remaining = value;

  for (let divisor = 2; divisor * divisor <= remaining; divisor += 1) {
    while (remaining % divisor === 0) {
      factors.push(divisor);
      remaining = Math.floor(remaining / divisor);
    }
  }

  if (remaining > 1) {
    factors.push(remaining);
  }

  return factors;
}
```

```ts

  return factors;
}
```

## Follow-up Questions

- Return compressed pairs of `(prime, exponent)` instead of a flat list.
- Use a smallest-prime-factor sieve to answer many factorizations quickly.
- Explain how the factorization feeds directly into Euler's totient formula.

## Common Pitfalls

- Dividing by a factor only once and losing repeated multiplicity.
- Forgetting to append the final leftover prime factor.
- Continuing to loop against the original `n` instead of the shrinking remainder.
