## Why It Surfaces

This is a small string-scan problem, but it is useful because the solution should stay proportional to the input length and avoid overcomplicating a fixed membership test.

## Worked Example

For `"Hello, World"`, the vowels are `e`, `o`, and `o`, so the count is `3`.

## Reference Implementations

<ExamplePair>

```js
function countVowels(value) {
  if (typeof value !== 'string') {
    throw new TypeError('Value must be a string');
  }

  let count = 0;

  for (const char of value.toLowerCase()) {
    if ('aeiou'.includes(char)) {
      count += 1;
    }
  }

  return count;
}
```

```ts
export function countVowels(value: string): number {
  let count = 0;

  for (const char of value.toLowerCase()) {
    if ('aeiou'.includes(char)) {
      count += 1;
    }
  }

  return count;
}
```

## Follow-up Questions

- Treat `y` as a vowel under a different rule set.
- Return separate counts for each vowel instead of a single total.
- Compare a regex-based implementation with explicit character scanning.

## Common Pitfalls

- Forgetting case normalization.
- Counting non-letters that happen to resemble vowels under other encodings.
- Reaching for heavier text-processing tools when a simple linear scan is enough.
