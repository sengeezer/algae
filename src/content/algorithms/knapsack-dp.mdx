## Why It Surfaces

Knapsack is the classic dynamic-programming frame for choosing the best subset under a hard limit. The story changes from backpacks to budgets, staffing, or time allocation, but the state transition stays the same.

## JavaScript

```js
function knapsack(weights, values, capacity) {
  const dp = new Array(capacity + 1).fill(0);

  for (let item = 0; item < weights.length; item += 1) {
    for (let space = capacity; space >= weights[item]; space -= 1) {
      dp[space] = Math.max(dp[space], dp[space - weights[item]] + values[item]);
    }
  }

  return dp[capacity];
}
```

## Interview Notes

- Iterate capacity backward so an item is not reused in the same row.
- Explain the difference between 0/1 knapsack and complete knapsack.
- If asked for the chosen items, mention reconstructing the path from a 2D table.
