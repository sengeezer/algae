## Why It Surfaces

This is a compact algebra question that checks whether you remember to classify the discriminant before blindly applying the quadratic formula.

## Worked Example

For `2x^2 - 4x + 2 = 0`, the discriminant is `(-4)^2 - 4 * 2 * 2 = 0`, so there is one repeated real root: `x = 1`. If the discriminant were negative, the routine would return no real roots.

## Reference Implementations

<ExamplePair>

```js
function quadraticRoots(a, b, c) {
  if (![a, b, c].every((value) => typeof value === 'number' && Number.isFinite(value))) {
    throw new TypeError('Coefficients must be finite numbers');
  }

  if (a === 0) {
    throw new RangeError('Leading coefficient must be nonzero');
  }

  const discriminant = b * b - 4 * a * c;
  if (discriminant < 0) {
    return [];
  }

  if (discriminant === 0) {
    return [-b / (2 * a)];
  }

  const sqrtDiscriminant = Math.sqrt(discriminant);
  return [
    (-b + sqrtDiscriminant) / (2 * a),
    (-b - sqrtDiscriminant) / (2 * a),
  ];
}
```

## Follow-up Questions

- Extend the solver to return complex roots when the discriminant is negative.
- Explain why `a = 0` is no longer a quadratic equation.
- Connect the roots back to the factorization of the corresponding polynomial.

## Common Pitfalls

- Forgetting to reject `a = 0`.
- Computing the square root before checking whether the discriminant is negative.
- Returning two identical roots instead of a single repeated root when the discriminant is zero.
