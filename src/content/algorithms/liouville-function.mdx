## Why It Surfaces

The Liouville function is a compact arithmetic helper that tests whether you count prime factors with multiplicity rather than only distinct factors.

## Worked Example

`12 = 2 * 2 * 3`, so it has `3` prime factors counted with multiplicity and $\lambda(12) = -1$. `36 = 2 * 2 * 3 * 3` has `4` such factors, so $\lambda(36) = 1$.

## Reference Implementations

<ExamplePair>

```js
function liouvilleFunction(value) {
  if (!Number.isInteger(value) || value <= 0) {
    throw new RangeError('Value must be a positive integer');
  }

  let remaining = value;
  let factorCount = 0;

  for (let divisor = 2; divisor * divisor <= remaining; divisor += 1) {
    while (remaining % divisor === 0) {
      factorCount += 1;
      remaining /= divisor;
    }
  }

  if (remaining > 1) {
    factorCount += 1;
  }

  return factorCount % 2 === 0 ? 1 : -1;
}
```

## Common Pitfalls

- Counting only distinct prime factors.
- Accepting zero or negative inputs without defining the function there.
- Factoring naively all the way to `n` instead of stopping near the square root.
