## Why It Surfaces

This is the directed counterpart to generic graph cloning, but the DAG constraint changes how the problem is framed in interviews. The graph has no cycles, yet memoization still matters because multiple parents can share the same descendant subtree.

## Worked Example

If node `0` points to `1` and `2`, and both `1` and `2` point to `3`, the clone must create only one cloned copy of `3`. A DFS with an original-to-clone map preserves that sharing instead of duplicating the same descendant once per parent.

## Reference Implementations

<ExamplePair>

```js
class DagNode {
  constructor(value) {
    this.value = value;
    this.neighbors = [];
  }
}

function cloneADirectedAcyclicGraph(node, clones = new Map()) {
  if (node === null) {
    return null;
  }

  if (clones.has(node)) {
    return clones.get(node);
  }

  const clone = new DagNode(node.value);
  clones.set(node, clone);

  for (const neighbor of node.neighbors) {
    clone.neighbors.push(cloneADirectedAcyclicGraph(neighbor, clones));
  }

  return clone;
}
```

```ts
class DagNode {
  value: number;
  neighbors: DagNode[];

  constructor(value: number) {
    this.value = value;
    this.neighbors = [];
  }
}

export function cloneADirectedAcyclicGraph(
  node: DagNode | null,
  clones: Map<DagNode, DagNode> = new Map<DagNode, DagNode>(),
): DagNode | null {
  if (node === null) {
    return null;
  }

  if (clones.has(node)) {
    return clones.get(node)!;
  }

  const clone = new DagNode(node.value);
  clones.set(node, clone);

  for (const neighbor of node.neighbors) {
    clone.neighbors.push(cloneADirectedAcyclicGraph(neighbor, clones)!);
  }

  return clone;
}
```

## Follow-up Questions

- How would the solution change if the graph could contain cycles?
- How would you clone all components when no single start node reaches every vertex?
- What test would prove that shared descendants were preserved correctly?

## Common Pitfalls

- Assuming a DAG can be cloned without a memo table because there are no cycles.
- Reusing original neighbor references and producing a shallow copy.
- Creating duplicate clones for shared descendants.
