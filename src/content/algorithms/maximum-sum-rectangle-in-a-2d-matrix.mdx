## Why It Surfaces

This is the standard way to show that Kadane's algorithm is bigger than one dimension. Once two column boundaries are fixed, every valid rectangle inside that strip becomes a contiguous range of compressed row sums, so the 2D problem reduces to the 1D maximum-subarray problem.

## Worked Example

For

```txt
1   2  -1  -4  -20
-8 -3   4   2    1
3   8  10   1    3
-4 -1   1   7   -6
```

the answer is `29`, produced by the rectangle covering the middle block with row sums driven by `3 + 8 + 10 + 1 + 3 + 7`.

## Reference Implementations

<ExamplePair>

```js
function maximumSumRectangleInA2DMatrix(matrix) {
  const rowCount = matrix.length;
  const colCount = rowCount === 0 ? 0 : matrix[0].length;
  if (rowCount === 0 || colCount === 0) {
    return 0;
  }

  function kadane(values) {
    let current = values[0];
    let best = values[0];

    for (let index = 1; index < values.length; index += 1) {
      current = Math.max(values[index], current + values[index]);
      best = Math.max(best, current);
    }

    return best;
  }

  let best = Number.NEGATIVE_INFINITY;

  for (let left = 0; left < colCount; left += 1) {
    const compressed = new Array(rowCount).fill(0);

    for (let right = left; right < colCount; right += 1) {
      for (let row = 0; row < rowCount; row += 1) {
        compressed[row] += matrix[row][right];
      }

      best = Math.max(best, kadane(compressed));
    }
  }

  return best;
}
```

  const colCount = rowCount === 0 ? 0 : matrix[0].length;
  if (rowCount === 0 || colCount === 0) {
    return 0;
  }

  function kadane(values: number[]): number {
    let current = values[0];
    let best = values[0];

    for (let index = 1; index < values.length; index += 1) {
      current = Math.max(values[index], current + values[index]);
      best = Math.max(best, current);
    }

    return best;
  }

  let best = Number.NEGATIVE_INFINITY;

    const compressed = new Array<number>(rowCount).fill(0);

      for (let row = 0; row < rowCount; row += 1) {
        compressed[row] += matrix[row][right];
      }

      best = Math.max(best, kadane(compressed));
    }
  }

  return best;
}

## Common Pitfalls

- Reusing a Kadane variant that incorrectly returns `0` for all-negative input.
- Compressing rows once and forgetting to reset them when the left boundary moves.
- Searching arbitrary cell subsets instead of contiguous rectangles.
