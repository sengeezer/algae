## Why It Surfaces

Arbitrary base conversion is the general case behind all the familiar binary, decimal, and hexadecimal helpers. Once you can decode a digit string with one alphabet and rebuild it with another, every fixed-base converter becomes a small specialization of the same idea.

## Worked Example

To convert `1011` from binary to hexadecimal, first decode it as `11` in decimal, then re-encode that value with the hex alphabet to get `B`. The same routine works for custom alphabets like base36 or base62.

```js
function assertUniqueCharacters(characters) {
  if (characters.length !== new Set(characters).size) {
    throw new TypeError('Duplicate characters in an alphabet are not allowed');
  }
}

function convertArbitraryBase(input, sourceAlphabet, targetAlphabet) {
  if ([input, sourceAlphabet, targetAlphabet].some((value) => typeof value !== 'string')) {
    throw new TypeError('Only string arguments are allowed');
  }

  const sourceDigits = [...sourceAlphabet];
  const targetDigits = [...targetAlphabet];

  if (sourceDigits.length < 2 || targetDigits.length < 2) {
    throw new RangeError('Each alphabet must contain at least two unique digits');
  }

  assertUniqueCharacters(sourceDigits);
  assertUniqueCharacters(targetDigits);

  let value = 0n;
  const sourceBase = BigInt(sourceDigits.length);

  for (const digit of input) {
    const digitValue = sourceDigits.indexOf(digit);
    if (digitValue === -1) {
      throw new TypeError(`Not a valid character: ${digit}`);
    }

    value = value * sourceBase + BigInt(digitValue);
  }

  if (value === 0n) {
    return targetDigits[0];
  }

  const targetBase = BigInt(targetDigits.length);
  const output = [];

  while (value > 0n) {
    const remainder = value % targetBase;
    output.push(targetDigits[Number(remainder)]);
    value /= targetBase;
  }

  return output.reverse().join('');
}
```

## Common Pitfalls

- Allowing duplicate characters in the source or target alphabet.
- Forgetting that `0` should still produce one output digit.
- Relying on built-in integer parsers that only support a few standard bases.
