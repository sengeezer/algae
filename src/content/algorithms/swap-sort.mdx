## Why It Surfaces

This problem looks like sorting, but the output is really a count. The trick is to know where every value belongs in the sorted array and then swap each misplaced value directly into that destination.

## Worked Example

For `[4, 3, 2, 1]`, the sorted order is `[1, 2, 3, 4]`. Swap `4` with `1` to get `[1, 3, 2, 4]`, then swap `3` with `2` to get `[1, 2, 3, 4]`. The minimum swap count is `2`.

## Reference Implementations

<ExamplePair>

```js
function minSwapsToSort(values) {
  if (new Set(values).size !== values.length) {
    throw new RangeError('Values must be distinct for this swap-counting version');
  }

  const working = [...values];
  const sorted = [...values].sort((left, right) => left - right);
  const indexByValue = new Map(working.map((value, index) => [value, index]));
  let swaps = 0;

  for (let index = 0; index < working.length; index += 1) {
    const desired = sorted[index];

    if (working[index] === desired) {
      continue;
    }

    const swapIndex = indexByValue.get(desired);
    const current = working[index];
    [working[index], working[swapIndex]] = [working[swapIndex], working[index]];
    indexByValue.set(current, swapIndex);
    indexByValue.set(desired, index);
    swaps += 1;
  }

  return swaps;
}
```

```ts
export function minSwapsToSort(values: number[]): number {
  if (new Set(values).size !== values.length) {
    throw new RangeError('Values must be distinct for this swap-counting version');
  }

  const working = [...values];
  const sorted = [...values].sort((left, right) => left - right);
  const indexByValue = new Map<number, number>(
    working.map((value, index) => [value, index]),
  );
  let swaps = 0;

  for (let index = 0; index < working.length; index += 1) {
    const desired = sorted[index];

    if (working[index] === desired) {
      continue;
    }

    const swapIndex = indexByValue.get(desired)!;
    const current = working[index];
    [working[index], working[swapIndex]] = [working[swapIndex], working[index]];
    indexByValue.set(current, swapIndex);
    indexByValue.set(desired, index);
    swaps += 1;
  }

  return swaps;
}
```

## Follow-up Questions

- Adapt the solution for duplicate values by sorting `(value, originalIndex)` pairs.
- Explain how cycle decomposition gives the same swap count.
- Return the actual swap operations instead of only the total.

## Common Pitfalls

- Using JavaScript's default lexical sort instead of a numeric comparator.
- Forgetting that the simple index map breaks when duplicate values exist.
- Mutating the caller's array when you only need the swap count.
