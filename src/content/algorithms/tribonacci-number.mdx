## Why It Surfaces

Tribonacci is the classic recurrence follow-up after Fibonacci. It tests whether you understand the DP pattern itself, not just the memorized formula for one specific sequence.

## Worked Example

With base values `T(0) = 0`, `T(1) = 1`, and `T(2) = 1`, the next value is `T(3) = 0 + 1 + 1 = 2`. Every later state reuses the previous three totals in the same way.

## Reference Implementations

<ExamplePair>

```js
function tribonacciNumber(index) {
  if (index === 0) {
    return 0;
  }

  if (index <= 2) {
    return 1;
  }

  const dp = new Array(index + 1).fill(0);
  dp[1] = 1;
  dp[2] = 1;

  for (let current = 3; current <= index; current += 1) {
    dp[current] = dp[current - 1] + dp[current - 2] + dp[current - 3];
  }

  return dp[index];
}
```

```ts
export function tribonacciNumber(index: number): number {
  if (index === 0) {
    return 0;
  }

  if (index <= 2) {
    return 1;
  }

  let first = 0;
  let second = 1;
  let third = 1;

  for (let current = 3; current <= index; current += 1) {
    const next = first + second + third;
    first = second;
    second = third;
    third = next;
  }

  return third;
}
```

## Common Pitfalls

- Reusing Fibonacci base cases without adjusting them for Tribonacci.
- Writing exponential recursion when iterative DP is enough.
- Forgetting that `T(0)` is typically `0`, not `1`.
