## Why It Surfaces

This is a formula-recognition warm-up in the quadrilateral sequence. Because opposite sides of a parallelogram are equal, the total boundary length is just twice the sum of the two adjacent side lengths.

## Worked Example

For side lengths `10` and `8`, the circumference is `2 * (10 + 8) = 36`.

## Reference Implementations

<ExamplePair>

```js
function programForCircumferenceOfAParallelogram(a, b) {
  if (!Number.isFinite(a) || !Number.isFinite(b) || a < 0 || b < 0) {
    throw new RangeError("Side lengths must be non-negative numbers");
  }

  return 2 * (a + b);
}
```

```ts
export function programForCircumferenceOfAParallelogram(a: number, b: number): number {
  if (!Number.isFinite(a) || !Number.isFinite(b) || a < 0 || b < 0) {
    throw new RangeError('Side lengths must be non-negative numbers');
  }

  return 2 * (a + b);
}
```

## Common Pitfalls

- Forgetting to double both adjacent sides.
- Confusing circumference with area.
- Assuming the shape must be a rectangle because only two side lengths are provided.
