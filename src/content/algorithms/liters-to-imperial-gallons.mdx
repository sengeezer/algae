## Why It Surfaces

This is the simplest kind of conversion problem: one quantity, one factor, one result. It is still worth cataloging because these small helpers show up constantly in product code and still need clear unit definitions.

## Worked Example

`9.09218` liters becomes `2` imperial gallons because one imperial gallon is `4.54609` liters.

## Reference Implementations

<ExamplePair>

```js
function litersToImperialGallons(liters) {
  if (!Number.isFinite(liters)) {
    throw new TypeError('Liters must be a finite number');
  }

  return liters / 4.54609;
}
```

```ts
export function litersToImperialGallons(liters: number): number {
  if (!Number.isFinite(liters)) {
    throw new TypeError('Liters must be a finite number');
  }

  return liters / 4.54609;
}
```
