## Why It Surfaces

Aliquot sum is a compact number-theory exercise that tests whether you can enumerate divisors efficiently and handle edge cases like `1` cleanly.

## Worked Example

For `12`, the proper divisors are `1`, `2`, `3`, `4`, and `6`, so the aliquot sum is `16`. For `6`, the proper divisors sum back to `6`, which makes it a perfect number.

## Reference Implementations

<ExamplePair>

```js
function aliquotSum(value) {
  if (!Number.isInteger(value) || value <= 0) {
    throw new RangeError('Value must be a positive integer');
  }

  if (value === 1) {
    return 0;
  }

  let total = 1;

  for (let divisor = 2; divisor * divisor <= value; divisor += 1) {
    if (value % divisor !== 0) {
      continue;
    }

    total += divisor;
    const paired = value / divisor;
    if (paired !== divisor) {
      total += paired;
    }
  }

  return total;
}
```

## Follow-up Questions

- Use aliquot sums to classify numbers as perfect, abundant, or deficient.
- Compare the square-root approach with a naive full scan.
- Cache divisor sums when evaluating many consecutive numbers.

## Common Pitfalls

- Counting the number itself as a proper divisor.
- Double-counting the square root for perfect squares.
- Doing an `O(n)` scan when a divisor-pair approach is available.
