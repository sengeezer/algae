## Why It Surfaces

This is a formula-recognition prompt in the quadrilateral section. The main detail is that the area depends on the two parallel sides and the perpendicular height, while the perimeter uses all four sides, so the helper needs both groups of inputs.

## Worked Example

For `a = 5`, `b = 6`, `c = 4`, `d = 3`, and `h = 8`, the area is `0.5 * (5 + 6) * 8 = 44` and the perimeter is `5 + 6 + 4 + 3 = 18`.

function programToCalculateAreaAndPerimeterOfTrapezium(a, b, c, d, h) {
  const values = [a, b, c, d, h];

    perimeter: a + b + c + d,

## Common Pitfalls

- Forgetting that perimeter needs all four sides, not just the bases.
- Using the average of all four sides in the area formula.
- Treating the height as a slanted side length instead of the perpendicular distance between bases.
