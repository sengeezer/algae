## Why It Surfaces

This is the four-term version of the familiar Pythagorean identity. Geometrically, it checks whether a cuboid with integer side lengths also has an integer space diagonal.

## Worked Example

`(1, 2, 2, 3)` is a Pythagorean quadruple because `1^2 + 2^2 + 2^2 = 9 = 3^2`.

## Reference Implementations

<ExamplePair>

```js
function pythagoreanQuadruple(a, b, c, d) {
  const values = [a, b, c, d];
  if (!values.every(Number.isInteger)) {
    throw new RangeError("Pythagorean quadruple inputs must be integers");
  }

  const [aa, bb, cc, dd] = values.map((value) => Math.abs(value));
  return aa ** 2 + bb ** 2 + cc ** 2 === dd ** 2;
}
```

```ts
export function pythagoreanQuadruple(
  a: number,
  b: number,
  c: number,
  d: number,
): boolean {
  const values = [a, b, c, d];
  if (!values.every(Number.isInteger)) {
    throw new RangeError('Pythagorean quadruple inputs must be integers');
  }

  const [aa, bb, cc, dd] = values.map((value) => Math.abs(value));
  return aa ** 2 + bb ** 2 + cc ** 2 === dd ** 2;
}
```

## Common Pitfalls

- Forgetting that the relation is defined on integers, not arbitrary floats.
- Comparing unsquared values or only two of the three left-hand terms.
- Rejecting negative integers even though squaring removes the sign.
