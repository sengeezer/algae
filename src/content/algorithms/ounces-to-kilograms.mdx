## Why It Surfaces

This is another direct unit formula, but for mass instead of distance or volume. It is useful because it reinforces that many production conversions are tiny, exact helpers that still benefit from explicit units and validation.

## Worked Example

`35.27396` ounces is about `1` kilogram. The conversion scales ounces into grams and then normalizes grams into kilograms.

## Reference Implementations

<ExamplePair>

```js
function ouncesToKilograms(ounces) {
  if (!Number.isFinite(ounces)) {
    throw new TypeError('Ounces must be a finite number');
  }

  return (ounces * 28.3498) / 1000;
}
```

```ts
export function ouncesToKilograms(ounces: number): number {
  if (!Number.isFinite(ounces)) {
    throw new TypeError('Ounces must be a finite number');
  }

  return (ounces * 28.3498) / 1000;
}
```

## Common Pitfalls

- Mixing avoirdupois ounces with other ounce definitions.
- Returning rounded strings when callers still expect numeric output.
- Treating the factor as exact when the surrounding domain may specify a preferred precision.
