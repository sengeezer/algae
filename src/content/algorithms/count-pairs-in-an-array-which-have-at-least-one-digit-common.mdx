## Why It Surfaces

This is the kind of counting problem that looks quadratic until you notice the domain is only decimal digits. Once each number becomes a 10-bit mask, the question changes from "Do these two numbers share a digit?" to "Do these two masks share a set bit?"

## Worked Example

For `[10, 12, 24]`, the valid pairs are `(10, 12)` because they both contain digit `1`, and `(12, 24)` because they both contain digit `2`, so the answer is `2`.

function digitMask(value) {
  if (value === 0) {
    return 1;
  }

  let number = value;
  let mask = 0;

  while (number > 0) {
    mask |= 1 << (number % 10);
    number = Math.floor(number / 10);
  }

  return mask;
}

function countPairsInAnArrayWhichHaveAtLeastOneDigitCommon(values) {
  if (!Array.isArray(values) || !values.every((value) => Number.isSafeInteger(value) && value >= 0)) {
    throw new TypeError('Values must be an array of non-negative safe integers');
  }

  const frequencies = new Array(1 << 10).fill(0);

  for (const value of values) {
    frequencies[digitMask(value)] += 1;
  }

  let pairs = 0;

  for (let leftMask = 1; leftMask < frequencies.length; leftMask += 1) {
    const leftFrequency = frequencies[leftMask];
    if (leftFrequency === 0) {
      continue;
    }

    pairs += (leftFrequency * (leftFrequency - 1)) / 2;

    for (let rightMask = leftMask + 1; rightMask < frequencies.length; rightMask += 1) {
      const rightFrequency = frequencies[rightMask];
      if (rightFrequency === 0) {
        continue;
      }

      if ((leftMask & rightMask) !== 0) {
        pairs += leftFrequency * rightFrequency;
      }
    }
  }

  return pairs;
}

## Follow-up Questions

- Return the actual pairs for small inputs, not only their count.
- Generalize the mask idea to hexadecimal digits or lowercase letters.
- Replace the `1024^2` mask scan with a subset-sum style acceleration if the alphabet grew larger.

## Common Pitfalls

- Forgetting that the number `0` contains digit `0` and therefore needs a non-empty mask.
- Using bitwise OR instead of bitwise AND when testing whether two masks share a digit.
- Building masks from decimal strings and accidentally treating repeated digits as multiple distinct requirements.
