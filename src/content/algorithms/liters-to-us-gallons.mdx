## Why It Surfaces

This helper looks almost identical to the imperial-gallon version, which is exactly why it matters. Near-identical conversions can still be wrong if the unit definition changes.

## Worked Example

`7.570823568` liters becomes `2` US gallons because one US gallon is `3.785411784` liters.

## Reference Implementations

<ExamplePair>

```js
function litersToUsGallons(liters) {
  if (!Number.isFinite(liters)) {
    throw new TypeError('Liters must be a finite number');
  }

  return liters / 3.785411784;
}
```

```ts
export function litersToUsGallons(liters: number): number {
  if (!Number.isFinite(liters)) {
    throw new TypeError('Liters must be a finite number');
  }

  return liters / 3.785411784;
}
```

- Treating unit conversion as locale formatting instead of arithmetic.
