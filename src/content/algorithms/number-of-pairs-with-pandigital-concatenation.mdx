## Why It Surfaces

This is a pair-count problem hiding inside a tiny bitmask universe. Once each numeric string becomes a 10-bit mask of seen digits, the only question left is whether two masks together cover all ten positions.

## Worked Example

For `['123567', '098234', '14765', '19804']`, there are `3` valid pairs. Each valid pair contributes all digits `0` through `9` at least once after concatenation.

function digitMask(value) {
  let mask = 0;

  for (const character of value) {
    mask |= 1 << Number.parseInt(character, 10);
  }

  return mask;
}

function numberOfPairsWithPandigitalConcatenation(values) {
  if (!Array.isArray(values) || !values.every((value) => typeof value === 'string' && /^[0-9]+$/.test(value))) {
    throw new TypeError('Values must be an array of non-empty decimal strings');
  }

  const fullMask = (1 << 10) - 1;
  const frequencies = new Array(1 << 10).fill(0);

  for (const value of values) {
    frequencies[digitMask(value)] += 1;
  }

  let pairs = 0;

  for (let leftMask = 1; leftMask < frequencies.length; leftMask += 1) {
    const leftFrequency = frequencies[leftMask];
    if (leftFrequency === 0) {
      continue;
    }

    if ((leftMask | leftMask) === fullMask) {
      pairs += (leftFrequency * (leftFrequency - 1)) / 2;
    }

    for (let rightMask = leftMask + 1; rightMask < frequencies.length; rightMask += 1) {
      const rightFrequency = frequencies[rightMask];
      if (rightFrequency === 0) {
        continue;
      }

      if ((leftMask | rightMask) === fullMask) {
        pairs += leftFrequency * rightFrequency;
      }
    }
  }

  return pairs;
}

## Follow-up Questions

- Return the actual pair indices instead of only the pair count.
- Generalize the mask logic to hexadecimal strings or lowercase letters.
- Compare this fixed-state-space method with brute-force concatenation on short inputs.

## Common Pitfalls

- Treating repeated digits inside one string as if they contributed extra coverage.
- Forgetting that two identical masks can still form valid pairs when that mask already covers all digits.
- Converting the strings to numbers and accidentally dropping significant leading zeros.
