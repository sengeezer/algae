## Why It Surfaces

Atbash is a straightforward mapping problem. The key interview move is preserving case and non-letter characters cleanly while noticing that the cipher is self-inverse.

## Worked Example

`"Algae"` becomes `"Zotzv"`: `A` mirrors to `Z`, `l` mirrors to `o`, `g` mirrors to `t`, and so on.

## Reference Implementations

<ExamplePair>

```js
function atbash(value) {
  if (typeof value !== 'string') {
    throw new TypeError('Value must be a string');
  }

  return value.replace(/[a-z]/gi, (char) => {
    const isUpper = /[A-Z]/.test(char);
    const base = isUpper ? 65 : 97;
    const index = char.charCodeAt(0) - base;
    return String.fromCharCode(base + (25 - index));
  });
}
```

```ts
export function atbash(value: string): string {
  return value.replace(/[a-z]/gi, (char) => {
    const isUpper = /[A-Z]/.test(char);
    const base = isUpper ? 65 : 97;
    const index = char.charCodeAt(0) - base;
    return String.fromCharCode(base + (25 - index));
  });
}
```

## Common Pitfalls

- Forgetting to preserve the case of the input characters.
- Touching digits or punctuation that should remain unchanged.
- Treating Atbash like a shift when it is really a mirror mapping.
