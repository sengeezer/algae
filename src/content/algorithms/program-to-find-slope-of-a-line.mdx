## Why It Surfaces

Slope is one of the first geometry-derived quantities that later problems reuse. Once you can compute rise over run cleanly and guard the vertical-line case, you have the building block for grouping collinear points or constructing line equations.

## Worked Example

For points `[4, 2]` and `[2, 5]`, the slope is `(5 - 2) / (2 - 4) = 3 / -2 = -1.5`.

## Reference Implementations

<ExamplePair>

```js
function programToFindSlopeOfALine(start, end) {
  const deltaX = end[0] - start[0];

  if (deltaX === 0) {
    return null;
  }

  return (end[1] - start[1]) / deltaX;
}
```

```ts
export function programToFindSlopeOfALine(
  start: [number, number],
  end: [number, number],
): number | null {
  const deltaX = end[0] - start[0];

  if (deltaX === 0) {
    return null;
  }

  return (end[1] - start[1]) / deltaX;
}
```

## Common Pitfalls

- Dividing by zero on vertical lines.
- Swapping rise and run.
- Assuming two equal slopes always mean the same line instead of only parallel direction.
