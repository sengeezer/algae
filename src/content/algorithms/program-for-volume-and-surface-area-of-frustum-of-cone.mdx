## Why It Surfaces

This prompt combines three standard frustum measurements. The main interview detail is keeping the roles of vertical height and slant height separate: volume uses the vertical height, while curved surface area uses the slant height.

## Worked Example

For small radius `3`, big radius `8`, height `12`, and slant height `13`, the volume is about `1218.9374`, the curved surface area is about `449.2477`, and the total surface area is about `678.5840`.

## Reference Implementations

```js
function programForVolumeAndSurfaceAreaOfFrustumOfCone(smallRadius, bigRadius, height, slantHeight) {
  const values = [smallRadius, bigRadius, height, slantHeight];
  if (!values.every(Number.isFinite) || values.some((value) => value < 0) || bigRadius < smallRadius) {
    throw new RangeError("Frustum dimensions must be non-negative, with big radius at least the small radius");
  }

  const volume = (Math.PI * height * (smallRadius * smallRadius + bigRadius * bigRadius + smallRadius * bigRadius)) / 3;
  const curvedSurfaceArea = Math.PI * slantHeight * (bigRadius + smallRadius);
  const totalSurfaceArea = curvedSurfaceArea + Math.PI * (smallRadius * smallRadius + bigRadius * bigRadius);

  return {
    volume,
    curvedSurfaceArea,
    totalSurfaceArea,
  };
}
```

## Common Pitfalls

- Mixing up the vertical height and the slant height.
- Omitting the areas of the two circular faces when computing total surface area.
- Reversing the big and small radii without validating the inputs.
