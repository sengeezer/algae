## Why It Surfaces

This is the ellipsoid analogue of the sphere-volume formula. The key observation is that the sphere's single radius is replaced by three independent semi-axes, so the volume scales by their product instead of by `r^3`.

## Worked Example

For semi-axes `2.3`, `3.4`, and `5.7`, the volume is about `186.711135`.

## Reference Implementations

<ExamplePair>

```js
function programToCalculateVolumeOfEllipsoid(axisA, axisB, axisC) {
  const axes = [axisA, axisB, axisC];
  if (!axes.every(Number.isFinite) || axes.some((axis) => axis < 0)) {
    throw new RangeError("Ellipsoid semi-axes must be non-negative numbers");
  }

  return (4 * Math.PI * axisA * axisB * axisC) / 3;
}
```

```ts
export function programToCalculateVolumeOfEllipsoid(
  axisA: number,
  axisB: number,
  axisC: number,
): number {
  const axes = [axisA, axisB, axisC];
  if (!axes.every(Number.isFinite) || axes.some((axis) => axis < 0)) {
    throw new RangeError('Ellipsoid semi-axes must be non-negative numbers');
  }

  return (4 * Math.PI * axisA * axisB * axisC) / 3;
}
```

## Common Pitfalls

- Forgetting the `4 / 3` factor.
- Using full axis lengths instead of semi-axis lengths.
- Treating negative axis lengths as valid input.
