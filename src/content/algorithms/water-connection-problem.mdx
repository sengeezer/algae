## Why It Surfaces

The greedy simplification comes from the graph constraint: each house belongs to at most one directed chain. That means every valid installation can be found by starting only at tank houses, then walking forward until the tap while tracking the smallest diameter seen so far.

## Worked Example

If the pipes are `1 -> 2 (30)`, `2 -> 3 (20)`, and `4 -> 5 (40)`, then houses `1` and `4` are tanks because they have outgoing pipes but no incoming pipes. The resulting installations are `[1, 3, 20]` and `[4, 5, 40]`.

<ExamplePair>

```js
function waterConnectionProblem(houseCount, from, to, diameters) {
  const outgoing = new Array(houseCount + 1).fill(0);
  const incoming = new Array(houseCount + 1).fill(0);
  const pipeDiameter = new Array(houseCount + 1).fill(0);

  for (let index = 0; index < from.length; index += 1) {
    outgoing[from[index]] = to[index];
    incoming[to[index]] = from[index];
    pipeDiameter[from[index]] = diameters[index];
  }

  const result = [];

  for (let house = 1; house <= houseCount; house += 1) {
    if (outgoing[house] === 0 || incoming[house] !== 0) {
      continue;
    }

    let current = house;
    let minimumDiameter = Number.POSITIVE_INFINITY;

    while (outgoing[current] !== 0) {
      minimumDiameter = Math.min(minimumDiameter, pipeDiameter[current]);
      current = outgoing[current];
    }

    result.push([house, current, minimumDiameter]);
  }

  return result;
}
```

## Follow-up Questions

- Detect malformed input where a house incorrectly has multiple outgoing or incoming pipes.
- Return the full chain of houses for each tank-tap connection instead of only the endpoints.
- Explain why starting the walk from every house would waste work once tanks are known.

## Common Pitfalls

- Starting from every node instead of only tank houses.
- Forgetting that the answer needs the minimum diameter on the chain, not the last pipe's diameter.
- Treating the input like a general graph even though the one-incoming-one-outgoing constraint makes the traversal linear.
