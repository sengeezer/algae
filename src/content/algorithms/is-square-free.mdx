## Why It Surfaces

Square-free checks are a good number-theory bridge between divisibility and factor structure. The efficient insight is that any repeated prime factor creates a square divisor.

## Worked Example

`30` is square-free because no square greater than `1` divides it. `12` is not square-free because `4` divides `12`.

## Reference Implementations

<ExamplePair>

```js
function isSquareFree(value) {
  if (!Number.isInteger(value) || value <= 0) {
    throw new RangeError('Value must be a positive integer');
  }

  for (let divisor = 2; divisor * divisor <= value; divisor += 1) {
    if (value % (divisor * divisor) === 0) {
      return false;
    }
  }

  return true;
}
```

## Common Pitfalls

- Allowing zero or negative values without defining the behavior.
- Factoring the whole number when checking square divisors is enough.
- Forgetting that a repeated prime factor always creates a square divisor.
