## Why It Surfaces

This is a direct formula question that still checks whether you remember the convergence rule for infinite series and the special case when the ratio is exactly `1`.

## Worked Example

For first term `2`, ratio `0.5`, and `6` terms, the sum is `2 * (0.5^6 - 1) / (0.5 - 1) = 3.9375`. For infinitely many terms with ratio `0.5`, the sum converges to `2 / (1 - 0.5) = 4`.

## Reference Implementations

<ExamplePair>

```js
function sumOfGeometricProgression(firstTerm, commonRatio, terms) {
  if (![firstTerm, commonRatio].every((value) => typeof value === 'number' && Number.isFinite(value))) {
    throw new TypeError('First term and common ratio must be finite numbers');
  }

  if (terms === Infinity) {
    if (Math.abs(commonRatio) < 1) {
      return firstTerm / (1 - commonRatio);
    }

    throw new RangeError('Infinite geometric progression diverges for this ratio');
  }

  if (!Number.isInteger(terms) || terms < 0) {
    throw new RangeError('Term count must be a non-negative integer or Infinity');
  }

  if (terms === 0) {
    return 0;
  }

  if (commonRatio === 1) {
    return firstTerm * terms;
  }

  return (firstTerm * (commonRatio ** terms - 1)) / (commonRatio - 1);
}
```

## Follow-up Questions

- Compare the direct formula with an explicit iterative sum.
- Explain why the infinite series diverges when `|ratio| >= 1`.
- Derive the finite-sum formula algebraically from `S - rS`.

## Common Pitfalls

- Forgetting the special case when the common ratio is exactly `1`.
- Accepting an infinite series that does not converge.
- Mixing up the finite-term and infinite-series formulas.
