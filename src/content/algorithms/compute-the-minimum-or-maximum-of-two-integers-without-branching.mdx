## Why It Surfaces

This is the compact version of branchless selection. The interesting part is not replacing an `if` for its own sake; it is showing that a boolean-style mask can select a value algebraically.

## Worked Example

For `15` and `6`, the minimum is `6` and the maximum is `15`, so the function returns `[6, 15]`.

## Reference Implementations

<ExamplePair>

```js
function computeTheMinimumOrMaximumOfTwoIntegersWithoutBranching(left, right) {
  if (!Number.isInteger(left) || !Number.isInteger(right)) {
    throw new TypeError('Inputs must be integers');
  }

  const minInt = -(2 ** 31);
  const maxInt = 2 ** 31 - 1;
  if ([left, right].some((value) => value < minInt || value > maxInt)) {
    throw new RangeError('Inputs must fit in a signed 32-bit integer');
  }

  const minimum = (right ^ ((left ^ right) & -Number(left < right))) | 0;
  const maximum = (left ^ ((left ^ right) & -Number(left < right))) | 0;

  return [minimum, maximum];
}
```
