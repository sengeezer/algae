## Why It Surfaces

Fenwick trees are the lightest-weight dynamic upgrade from prefix sums. Instead of storing one fixed prefix array, they keep overlapping buckets whose sizes come from the least significant set bit of each index.

## Worked Example

If index `12` stores the sum for the last `4` elements ending there, then moving to `12 - 4 = 8` peels off the next bucket while a query walks upward through the covered prefixes.

<Callout title="Bit trick" tone="tip">
  The lowest set bit controls both movement directions: subtract it when querying upward toward the root, add it when propagating an update to larger buckets.
</Callout>

## Reference Implementations

<ExamplePair>

```js
class FenwickTree {
  constructor(size) {
    this.tree = new Array(size + 1).fill(0);
  }

  update(index, delta) {
    let cursor = index + 1;

    while (cursor < this.tree.length) {
      this.tree[cursor] += delta;
      cursor += cursor & -cursor;
    }
  }

  query(index) {
    let cursor = index + 1;
    let total = 0;

    while (cursor > 0) {
      total += this.tree[cursor];
      cursor -= cursor & -cursor;
    }

    return total;
  }

  rangeQuery(left, right) {
    return this.query(right) - (left > 0 ? this.query(left - 1) : 0);
  }
}
```

```ts
  private readonly tree: number[];

  constructor(size: number) {
    this.tree = new Array<number>(size + 1).fill(0);
  }

  update(index: number, delta: number): void {
    let cursor = index + 1;

    while (cursor < this.tree.length) {
      this.tree[cursor] += delta;
      cursor += cursor & -cursor;
    }
  }

  query(index: number): number {
    let cursor = index + 1;
    let total = 0;

    while (cursor > 0) {
      total += this.tree[cursor];
      cursor -= cursor & -cursor;
    }

    return total;
  }

  rangeQuery(left: number, right: number): number {
    return this.query(right) - (left > 0 ? this.query(left - 1) : 0);
  }
}
```

## Interview Notes

- Say explicitly whether the external API is zero-indexed or one-indexed.
- Mention that Fenwick trees are best when the operation is invertible, such as sums or counts.
- If the prompt asks for more complex range operations, pivot to a segment tree.
