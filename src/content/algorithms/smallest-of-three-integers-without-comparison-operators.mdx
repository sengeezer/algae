## Why It Surfaces

This is the multi-input extension of branchless minimum tricks. The interview value is less about production style and more about showing that you can turn a sign-bit observation into a reusable primitive.

## Worked Example

For `2`, `5`, and `7`, the smallest value is `2`, so the function returns `2`. For `14`, `13`, and `10`, it returns `10`.

## Reference Implementations

<ExamplePair>

```js
function smallestOfThreeIntegersWithoutComparisonOperators(first, second, third) {
  if (![first, second, third].every(Number.isInteger)) {
    throw new TypeError('Inputs must be integers');
  }

  const minInt = -(2 ** 31);
  const maxInt = 2 ** 31 - 1;
  if ([first, second, third].some((value) => value < minInt || value > maxInt)) {
    throw new RangeError('Inputs must fit in a signed 32-bit integer');
  }

  function bitwiseMin(left, right) {
    const leftValue = BigInt(left);
    const rightValue = BigInt(right);
    const difference = leftValue - rightValue;
    const signMask = difference >> 63n;
    return Number(rightValue + (difference & signMask));
  }

  return bitwiseMin(first, bitwiseMin(second, third));
}
```

  first: number,
  second: number,
  third: number,
): number {
  if (![first, second, third].every(Number.isInteger)) {
    throw new TypeError('Inputs must be integers');
  }

  const minInt = -(2 ** 31);
  const maxInt = 2 ** 31 - 1;

    const difference = leftValue - rightValue;
    const signMask = difference >> 63n;
    return Number(rightValue + (difference & signMask));

## Follow-up Questions

- Extend the same idea to the largest of three numbers.
- Explain when the subtraction-based sign extraction can overflow in fixed-width languages.
- Compare this branchless trick with a normal `Math.min` implementation.

## Common Pitfalls

- Forgetting that subtraction-based sign extraction can break at signed 32-bit extremes if the intermediate difference is allowed to overflow.
- Solving only the two-number case and not composing it correctly for three inputs.
- Treating this interview trick as universally clearer than an ordinary comparison-based minimum.
