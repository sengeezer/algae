## Why It Surfaces

This prompt packages the two canonical sphere formulas together. The only real trick is not mixing up the quadratic surface-area formula with the cubic volume formula when both are driven by the same radius.

## Worked Example

For `radius = 5`, the volume is about `523.598776` and the surface area is about `314.159265`.

## Reference Implementations

<ExamplePair>

```js
function calculateVolumeAndSurfaceAreaOfSphere(radius) {
  if (!Number.isFinite(radius) || radius < 0) {
    throw new RangeError("Sphere radius must be a non-negative number");
  }

  return {
    volume: (4 * Math.PI * radius ** 3) / 3,
    surfaceArea: 4 * Math.PI * radius ** 2,
  };
}
```

```ts
export function calculateVolumeAndSurfaceAreaOfSphere(
  radius: number,
): { volume: number; surfaceArea: number } {
  if (!Number.isFinite(radius) || radius < 0) {
    throw new RangeError('Sphere radius must be a non-negative number');
  }

  return {
    volume: (4 * Math.PI * radius ** 3) / 3,
    surfaceArea: 4 * Math.PI * radius ** 2,
  };
}
```

## Common Pitfalls

- Forgetting the `4 / 3` factor in the volume formula.
- Reusing `r^3` in the surface-area formula.
- Treating a negative radius as a valid physical input.
