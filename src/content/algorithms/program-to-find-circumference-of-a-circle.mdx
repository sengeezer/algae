## Why It Surfaces

This is one of the simplest shape-formula prompts in the geometric catalog. It still matters because interviewers often pair it with area-of-a-circle or use it as a warm-up for checking whether the candidate handles direct formulas and input validation cleanly.

## Worked Example

For `radius = 5`, the circumference is `2 * pi * 5`, which is about `31.415927`.

## Reference Implementations

<ExamplePair>

```js
function programToFindCircumferenceOfACircle(radius) {
  if (typeof radius !== "number" || !Number.isFinite(radius) || radius < 0) {
    throw new RangeError("Radius must be a non-negative number");
  }

  return 2 * Math.PI * radius;
}
```

```ts
export function programToFindCircumferenceOfACircle(radius: number): number {
  if (!Number.isFinite(radius) || radius < 0) {
    throw new RangeError('Radius must be a non-negative number');
  }

  return 2 * Math.PI * radius;
}
```
