## Why It Surfaces

Cube prompts are the simplest combined solid-measurement exercise. Because every face is the same square, the formulas collapse to one power for volume and one constant-factor multiple of the squared side for area.

## Worked Example

For `side = 3`, the volume is `27` and the total surface area is `54`.

## Reference Implementations

<ExamplePair>

```js
function programForVolumeAndSurfaceAreaOfCube(side) {
  if (!Number.isFinite(side) || side < 0) {
    throw new RangeError("Cube side length must be a non-negative number");
  }

  return {
    volume: side ** 3,
    surfaceArea: 6 * side ** 2,
  };
}
```

```ts
export function programForVolumeAndSurfaceAreaOfCube(
  side: number,
): { volume: number; surfaceArea: number } {
  if (!Number.isFinite(side) || side < 0) {
    throw new RangeError('Cube side length must be a non-negative number');
  }

  return {
    volume: side ** 3,
    surfaceArea: 6 * side ** 2,
  };
}
```

## Common Pitfalls

- Using `6 * side ** 3` instead of `6 * side ** 2` for surface area.
- Copying the source article's mislabeled output without clarifying that the first value is really volume.
- Accepting negative side lengths as if they described a valid cube.
