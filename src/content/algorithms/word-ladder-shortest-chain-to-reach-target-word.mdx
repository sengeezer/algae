## Why It Surfaces

Word Ladder is the classic example of an implicit graph. Every valid dictionary word is a node, every one-letter mutation is an edge, and the interviewer wants the minimum number of steps, which makes breadth-first search the right fit immediately.

## Worked Example

If `toon` must become `plea` and the dictionary contains `poon`, `poin`, `poie`, `plie`, and `plee`, then BFS explores all one-step mutations before any two-step mutations. The first time it reaches `plea`, the chain length is guaranteed to be minimal.

```js
function wordLadderShortestChainToReachTargetWord(start, target, words) {
  if (start === target) {
    return 1;
  }

  if (start.length !== target.length) {
    return 0;
  }

  const dictionary = new Set(words);
  if (!dictionary.has(target)) {
    return 0;
  }

  dictionary.delete(start);
  const queue = [[start, 1]];

  for (let head = 0; head < queue.length; head += 1) {
    const [word, distance] = queue[head];
    const chars = word.split('');

    for (let index = 0; index < chars.length; index += 1) {
      const original = chars[index];

      for (let code = 97; code <= 122; code += 1) {
        const nextChar = String.fromCharCode(code);
        if (nextChar === original) {
          continue;
        }

        chars[index] = nextChar;
        const nextWord = chars.join('');

        if (!dictionary.has(nextWord)) {
          continue;
        }

        if (nextWord === target) {
          return distance + 1;
        }

        dictionary.delete(nextWord);
        queue.push([nextWord, distance + 1]);
      }

      chars[index] = original;
    }
  }

  return 0;
}
```

## Follow-up Questions

- Return the actual transformation sequence instead of only its length.
- Precompute wildcard buckets to reduce repeated neighbor generation.
- Explain why DFS or backtracking does not guarantee the shortest chain first.

## Common Pitfalls

- Treating the dictionary like a weighted graph problem when every valid transformation has unit cost.
- Forgetting to mark generated words as visited when they enter the queue.
- Returning the number of edges instead of the chain length that includes both endpoints.
