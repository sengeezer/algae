## Why It Surfaces

This problem is a good interview twist because it is not asking whether the whole graph has a cycle. Instead, it asks you to classify each connected component and count only the ones that form one clean loop. The decisive invariant is simple: inside an undirected connected component, every vertex must have degree `2` for that component to be exactly one cycle.

## Worked Example

With `vertexCount = 8` and edges `[[0, 1], [1, 2], [2, 0], [3, 4], [4, 5], [5, 6], [6, 3]]`, the graph contains a triangle and a four-cycle, plus isolated vertex `7`. The algorithm returns `2` because only the first two components are single cycles.

## Reference Implementations

<ExamplePair>

```js
function numberOfSingleCycleComponentsInAnUndirectedGraph(vertexCount, edges) {
  const adjacencyList = Array.from({ length: vertexCount }, () => []);

  for (const [from, to] of edges) {
    adjacencyList[from].push(to);
    adjacencyList[to].push(from);
  }

  const visited = Array(vertexCount).fill(false);
  let cycleComponents = 0;

  for (let start = 0; start < vertexCount; start += 1) {
    if (visited[start]) {
      continue;
    }

    const stack = [start];
    visited[start] = true;
    let isSingleCycle = true;

    while (stack.length > 0) {
      const node = stack.pop();

      if (adjacencyList[node].length !== 2) {
        isSingleCycle = false;
      }

      for (const neighbor of adjacencyList[node]) {
        if (!visited[neighbor]) {
          visited[neighbor] = true;
          stack.push(neighbor);
        }
      }
    }

    if (isSingleCycle) {
      cycleComponents += 1;
    }
  }

  return cycleComponents;
}
```

## Follow-up Questions

- Return the actual vertex lists for each cycle component instead of only the count.
- Adapt the logic to ignore isolated vertices or self-loops under a different graph model.
- Compare this traversal with a union-find pass that first groups connected components.

## Common Pitfalls

- Counting any component that contains a cycle instead of requiring every vertex to have degree `2`.
- Forgetting to process disconnected components separately.
- Treating isolated vertices or path components as valid cycles.
