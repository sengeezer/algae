## Why It Surfaces

This is a good example of geometry meeting number theory. Instead of scanning the whole bounding box, you compute the triangle's doubled area, count how many lattice points lie on each edge using gcd, and plug the result into Pick's theorem.

## Worked Example

For `P = [0, 0]`, `Q = [0, 5]`, and `R = [5, 0]`, Pick's theorem yields `6` interior integer points.

## Reference Implementations

<ExamplePair>

```js
function countIntegralPointsInsideATriangle(p, q, r) {
  function gcd(left, right) {
    let a = Math.abs(left);
    let b = Math.abs(right);

    while (b !== 0) {
      [a, b] = [b, a % b];
    }

    return a;
  }

  function edgeBoundaryCount(first, second) {
    return gcd(first[0] - second[0], first[1] - second[1]);
  }

  const doubledArea = Math.abs(
    p[0] * (q[1] - r[1])
    + q[0] * (r[1] - p[1])
    + r[0] * (p[1] - q[1])
  );

  if (doubledArea === 0) {
    return 0;
  }

  const boundaryPoints = (
    edgeBoundaryCount(p, q)
    + edgeBoundaryCount(q, r)
    + edgeBoundaryCount(r, p)
  );

  return (doubledArea - boundaryPoints + 2) / 2;
}
```

```ts
type Point = [number, number];

export function countIntegralPointsInsideATriangle(
  p: Point,
  q: Point,
  r: Point,
): number {
  const gcd = (left: number, right: number): number => {
    let a = Math.abs(left);
    let b = Math.abs(right);

    while (b !== 0) {
      [a, b] = [b, a % b];
    }

    return a;
  };

  const edgeBoundaryCount = (first: Point, second: Point): number => {
    return gcd(first[0] - second[0], first[1] - second[1]);
  };

  const doubledArea = Math.abs(
    p[0] * (q[1] - r[1])
    + q[0] * (r[1] - p[1])
    + r[0] * (p[1] - q[1])
  );

  if (doubledArea === 0) {
    return 0;
  }

  const boundaryPoints = (
    edgeBoundaryCount(p, q)
    + edgeBoundaryCount(q, r)
    + edgeBoundaryCount(r, p)
  );

  return (doubledArea - boundaryPoints + 2) / 2;
}
```

## Common Pitfalls

- Counting boundary lattice points incorrectly by double-counting vertices.
- Forgetting that Pick's theorem needs a simple polygon with lattice vertices.
- Enumerating every point in the bounding box instead of using the theorem directly.
