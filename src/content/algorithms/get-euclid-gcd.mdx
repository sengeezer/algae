## Why It Surfaces

This is one of the highest-value short routines in interview math. It underpins coprimality, modular inverses, fraction reduction, and many other helpers.

## Worked Example

For `gcd(84, 30)`, the Euclidean sequence is `84 % 30 = 24`, `30 % 24 = 6`, and `24 % 6 = 0`, so the gcd is `6`.

## Reference Implementations

<ExamplePair>

```js
function validateIntegers(left, right) {
  if (![left, right].every((value) => Number.isInteger(value))) {
    throw new TypeError('Arguments must be integers');
  }
}

function getEuclidGcd(left, right) {
  validateIntegers(left, right);

  let a = Math.abs(left);
  let b = Math.abs(right);

  while (b !== 0) {
    const remainder = a % b;
    a = b;
    b = remainder;
  }

  return a;
}

function getEuclidGcdRecursive(left, right) {
  validateIntegers(left, right);

  const a = Math.abs(left);
  const b = Math.abs(right);
  if (b === 0) {
    return a;
  }

  return getEuclidGcdRecursive(b, a % b);
}
```

## Common Pitfalls

- Factoring both numbers instead of using remainders.
- Forgetting to normalize signs with absolute values.
- Using the Euclidean algorithm on non-integer inputs without defining the behavior.
