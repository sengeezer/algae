## Why It Surfaces

After insert, search, and delete, the next natural trie question is usually enumeration. The key move is to carry the current prefix down the recursion and emit it only when the current node marks the end of a stored word.

## Worked Example

If the trie stores `answer`, `any`, `bye`, `their`, and `there`, a lexicographic DFS over child edges produces exactly that ordered list of words from the structure.

```js
class DisplayTrieNode {
  constructor() {
    this.children = new Map();
    this.isWord = false;
  }
}

class DisplayTrie {
  constructor() {
    this.root = new DisplayTrieNode();
  }

  insert(word) {
    if (typeof word !== 'string' || word.length === 0) {
      throw new RangeError('Words must be non-empty strings');
    }

    let node = this.root;
    for (const char of word) {
      if (!node.children.has(char)) {
        node.children.set(char, new DisplayTrieNode());
      }
      node = node.children.get(char);
    }
    node.isWord = true;
  }

  listContents() {
    const contents = [];

    const dfs = (node, prefix) => {
      if (node.isWord) {
        contents.push(prefix);
      }

      const entries = [...node.children.entries()].sort((left, right) =>
        left[0].localeCompare(right[0]),
      );

      for (const [char, child] of entries) {
        dfs(child, prefix + char);
      }
    };

    dfs(this.root, '');
    return contents;
  }
}
```

  isWord = false;
}

  private readonly root = new DisplayTrieNode();

  insert(word: string): void {
    if (word.length === 0) {
      throw new RangeError('Words must be non-empty strings');
    }

    let node = this.root;
    for (const char of word) {
      if (!node.children.has(char)) {
        node.children.set(char, new DisplayTrieNode());

      node = node.children.get(char) as DisplayTrieNode;

    node.isWord = true;

  }
}

## Common Pitfalls

- Printing prefixes that are not actually marked as complete words.
- Forgetting to preserve lexicographic order when iterating children.
- Reusing one mutable buffer without undoing the last appended character correctly.
