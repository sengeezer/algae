## Why It Surfaces

This is the matrix version of exponentiation by squaring. It matters because many recurrence and graph-transition problems become fast once you raise a matrix in logarithmic exponent time instead of linearly.

## Worked Example

Any square matrix raised to power `0` becomes the identity matrix of the same size. For power `2`, you multiply the matrix by itself once. For large exponents, repeated squaring keeps the work logarithmic in the power.

```js
function validateSquareMatrix(matrix) {
  if (!Array.isArray(matrix) || matrix.length === 0 || !matrix.every((row) => Array.isArray(row) && row.length === matrix.length)) {
    throw new RangeError('Input must be a non-empty square matrix');
  }
}

function identityMatrix(size) {
  return Array.from({ length: size }, (_, row) =>
    Array.from({ length: size }, (_, col) => (row === col ? 1 : 0)),
  );
}

function multiplySquareMatrices(left, right) {
  const size = left.length;
  const result = Array.from({ length: size }, () => new Array(size).fill(0));

  for (let row = 0; row < size; row += 1) {
    for (let col = 0; col < size; col += 1) {
      for (let index = 0; index < size; index += 1) {
        result[row][col] += left[row][index] * right[index][col];
      }
    }
  }

  return result;
}

function matrixExponentiationRecursive(matrix, power) {
  validateSquareMatrix(matrix);

  if (!Number.isInteger(power) || power < 0) {
    throw new RangeError('Power must be a non-negative integer');
  }

  if (power === 0) {
    return identityMatrix(matrix.length);
  }

  if (power === 1) {
    return matrix.map((row) => row.slice());
  }

  if (power % 2 === 1) {
    return multiplySquareMatrices(matrixExponentiationRecursive(matrix, power - 1), matrix);
  }

  const half = matrixExponentiationRecursive(matrix, power / 2);
  return multiplySquareMatrices(half, half);
}
```

## Common Pitfalls

- Forgetting that only square matrices can be raised to integer powers this way.
- Returning the original matrix instead of an identity matrix at power `0`.
- Multiplying linearly `p` times instead of squaring recursively.
