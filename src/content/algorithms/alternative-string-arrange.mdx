## Why It Surfaces

This is a clean exercise in synchronized traversal. The interesting part is not the alternation itself, but handling uneven string lengths without dropping leftover characters.

## Worked Example

Interleaving `"abc"` and `"XYZ12"` produces `"aXbYcZ12"`: take one character from each string while both have remaining characters, then append the rest of the longer input.

## Reference Implementations

<ExamplePair>

```js
function alternativeStringArrange(left, right) {
  if (typeof left !== 'string' || typeof right !== 'string') {
    throw new TypeError('Both arguments must be strings');
  }

  let result = '';
  const length = Math.max(left.length, right.length);

  for (let index = 0; index < length; index += 1) {
    if (index < left.length) {
      result += left[index];
    }

    if (index < right.length) {
      result += right[index];
    }
  }

  return result;
}
```

## Follow-up Questions

- Generalize to interleave $k$ strings round-robin.
- Merge by groups of two or three characters at a time instead of one.
- Build the result with an array buffer instead of repeated string concatenation.

## Common Pitfalls

- Stopping when the shorter string ends and dropping the remainder.
- Reversing the relative order of characters within one input.
- Returning sentinel strings instead of throwing or documenting invalid input behavior.
