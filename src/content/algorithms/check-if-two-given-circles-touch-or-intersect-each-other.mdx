## Why It Surfaces

This is a small but useful geometry classifier. The direct formula for center distance is familiar, but the stronger interview move is to compare squared values and cleanly split the cases into coincident, contained, tangent, intersecting, or disjoint.

## Worked Example

Circle `A = (0, 0, 5)` and circle `B = (6, 0, 4)` intersect because the center distance is `6`, which is smaller than `5 + 4` and larger than `|5 - 4|`.

## Reference Implementations

<ExamplePair>

```js
function checkIfTwoGivenCirclesTouchOrIntersectEachOther(circleA, circleB) {
  const values = [
    circleA?.x,
    circleA?.y,
    circleA?.radius,
    circleB?.x,
    circleB?.y,
    circleB?.radius,
  ];

  if (!values.every(Number.isFinite) || circleA.radius < 0 || circleB.radius < 0) {
    throw new RangeError("Circle centers and radii must be finite, with non-negative radii");
  }

  const dx = circleA.x - circleB.x;
  const dy = circleA.y - circleB.y;
  const distanceSq = dx * dx + dy * dy;
  const radiusSum = circleA.radius + circleB.radius;
  const radiusDiff = Math.abs(circleA.radius - circleB.radius);

  if (distanceSq === 0 && circleA.radius === circleB.radius) {
    return "coincident";
  }

  if (distanceSq < radiusDiff * radiusDiff) {
    return circleA.radius > circleB.radius ? "circle-b-inside-a" : "circle-a-inside-b";
  }

  if (distanceSq === radiusDiff * radiusDiff || distanceSq === radiusSum * radiusSum) {
    return "touch";
  }

  if (distanceSq < radiusSum * radiusSum) {
    return "intersect";
  }

  return "disjoint";
}
```

## Common Pitfalls

- Missing the contained-circle case and treating it as an intersection.
- Comparing floating distances for equality after taking a square root.
- Forgetting that equal centers and equal radii represent infinitely many intersection points.
