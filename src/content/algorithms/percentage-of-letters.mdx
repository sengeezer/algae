## Why It Surfaces

This is a compact way to combine counting with a tiny bit of arithmetic. The algorithm is simple, but it tests whether you can preserve the prompt’s exact rounding rule.

## Worked Example

For `"foobar"` and target `"o"`, the letter appears `2` times in a string of length `6`, so the answer is $\lfloor 100 \cdot 2 / 6 \rfloor = 33$.

## Reference Implementations

<ExamplePair>

```js
function percentageOfLetters(value, target) {
  if (typeof value !== 'string' || typeof target !== 'string') {
    throw new TypeError('Both arguments must be strings');
  }

  if (target.length !== 1) {
    throw new Error('Target must be exactly one character');
  }

  let count = 0;

  for (const char of value) {
    if (char === target) {
      count += 1;
    }
  }

  return value.length === 0 ? 0 : Math.floor((count * 100) / value.length);
}
```

## Follow-up Questions

- Make the comparison case-insensitive.
- Return the exact fraction instead of rounding down to a whole percent.
- Generalize the routine to percentages for all characters at once.

## Common Pitfalls

- Rounding to the nearest percent instead of flooring.
- Forgetting to define behavior for the empty string.
- Accepting a multi-character target without clarifying the contract.
