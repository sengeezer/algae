## Why It Surfaces

This prompt is distinct from a plain cone-volume helper because it combines two formulas with different inputs. The vertical height drives the volume, while the slant height drives the lateral part of the total surface area.

## Worked Example

For `radius = 5`, `slantHeight = 13`, and `height = 12`, the volume is about `314.159265` and the total surface area is about `282.743339`.

## Reference Implementations

<ExamplePair>

```js
function calculateVolumeAndSurfaceAreaOfACone(radius, slantHeight, height) {
  const values = [radius, slantHeight, height];
  if (!values.every(Number.isFinite) || values.some((value) => value < 0)) {
    throw new RangeError("Cone dimensions must be non-negative numbers");
  }

  return {
    volume: (Math.PI * radius * radius * height) / 3,
    surfaceArea: Math.PI * radius * slantHeight + Math.PI * radius * radius,
  };
}
```

```ts
export function calculateVolumeAndSurfaceAreaOfACone(
  radius: number,
  slantHeight: number,
  height: number,
): { volume: number; surfaceArea: number } {
  const values = [radius, slantHeight, height];
  if (!values.every(Number.isFinite) || values.some((value) => value < 0)) {
    throw new RangeError('Cone dimensions must be non-negative numbers');
  }

  return {
    volume: (Math.PI * radius * radius * height) / 3,
    surfaceArea: Math.PI * radius * slantHeight + Math.PI * radius * radius,
  };
}
```

## Follow-up Questions

## Common Pitfalls

- Reusing the vertical height in the surface-area formula.
- Forgetting to include the base area in the total surface area.
- Treating this as a duplicate of the volume-only cone helper.
