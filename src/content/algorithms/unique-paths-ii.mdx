## Why It Surfaces

This is the natural follow-up to Unique Paths. The recurrence stays almost identical, but obstacle cells break the propagation by contributing zero paths from that point onward.

## Worked Example

If the middle cell of a `3 x 3` grid is blocked, any route that would pass through it disappears. The final path count is just the sum of paths that reach the destination by going around that obstacle.

<Callout title="Reset rule" tone="tip">
  When a cell is blocked, set its path count to zero immediately. Do not try to add top and left values into an obstacle cell.
</Callout>

## Reference Implementations

<ExamplePair>

```js
function uniquePaths2(grid) {
  const rows = grid.length;
  const cols = grid[0].length;
  const dp = Array.from({ length: rows }, () => new Array(cols).fill(0));

  if (grid[0][0] === 1) {
    return 0;
  }

  dp[0][0] = 1;

  for (let row = 0; row < rows; row += 1) {
    for (let col = 0; col < cols; col += 1) {
      if (grid[row][col] === 1) {
        dp[row][col] = 0;
        continue;
      }

      if (row > 0) {
        dp[row][col] += dp[row - 1][col];
      }

      if (col > 0) {
        dp[row][col] += dp[row][col - 1];
      }
    }
  }

  return dp[rows - 1][cols - 1];
}
```

```ts
export function uniquePaths2(grid: number[][]): number {
  const cols = grid[0].length;
  const dp = new Array<number>(cols).fill(0);
  dp[0] = grid[0][0] === 1 ? 0 : 1;

  for (let row = 0; row < grid.length; row += 1) {
    for (let col = 0; col < cols; col += 1) {
      if (grid[row][col] === 1) {
        dp[col] = 0;
      } else if (col > 0) {
        dp[col] += dp[col - 1];
      }
    }
  }

  return dp[cols - 1];
}
```

## Follow-up Questions

- Compare the one-row DP with a full grid table.
- Return one valid route instead of the total count.
- Extend this to weighted grids where cost replaces count.

## Common Pitfalls

- Failing to zero out blocked cells.
- Forgetting the blocked-start or blocked-end edge cases.
- Mixing this counting problem up with shortest-path traversal.
