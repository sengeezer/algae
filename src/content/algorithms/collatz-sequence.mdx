## Why It Surfaces

Collatz is a simple recurrence with memorable rules, so it is a common prompt for practicing branching, iteration, and sequence construction on integers.

## Worked Example

Starting from `5`, the sequence goes `16, 8, 4, 2, 1`. Odd values jump to `3n + 1`, and even values halve immediately.

## Reference Implementations

<ExamplePair>

```js
function collatzSequence(value) {
  if (!Number.isInteger(value) || value <= 0) {
    throw new RangeError('Value must be a positive integer');
  }

  const steps = [];
  let current = value;

  while (current !== 1) {
    current = current % 2 === 0 ? current / 2 : 3 * current + 1;
    steps.push(current);
  }

  return { result: current, steps };
}
```

```ts

    steps.push(current);
  }

  return { result: current, steps };
}
```

## Follow-up Questions

- Return only the step count instead of the full sequence.
- Track the maximum value reached before the sequence hits `1`.
- Compare storing the sequence with streaming each generated value to a callback.

## Common Pitfalls

- Accepting zero or negative values even though the routine here is defined for positive integers.
- Forgetting to append the updated value after each step.
- Assuming the conjecture itself is proven when the coding task only needs the recurrence.
