## Why It Surfaces

Shell sort is a practical example of improving insertion sort without abandoning its core mechanic. By sorting elements that are far apart first, the final gap-1 insertion pass has much less work left to do.

## Worked Example

With the array `[23, 12, 1, 8, 34, 54, 2, 3]`, a large initial gap lets small values like `1` and `2` move toward the front early. Later passes with smaller gaps clean up the remaining local inversions.

## Reference Implementations

<ExamplePair>

```js
function shellSort(values) {
  if (!Array.isArray(values)) {
    throw new TypeError('Values must be an array of numbers');
  }

  const items = [...values];
  let gap = 1;

  while (gap < items.length / 3) {
    gap = gap * 3 + 1;
  }

  while (gap >= 1) {
    for (let index = gap; index < items.length; index += 1) {
      const value = items[index];
      let position = index;

      while (position >= gap && items[position - gap] > value) {
        items[position] = items[position - gap];
        position -= gap;
      }

      items[position] = value;
    }

    gap = Math.floor((gap - 1) / 3);
  }

  return items;
}
```

## Common Pitfalls

- Dropping straight into gap 1 and losing the algorithm's benefit.
- Updating the gap incorrectly and getting stuck in an infinite loop.
- Talking about a single fixed complexity without qualifying the gap sequence.
