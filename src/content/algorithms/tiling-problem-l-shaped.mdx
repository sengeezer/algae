## Why It Surfaces

This problem is memorable because the recursive step is constructive, not merely analytical. One central tromino turns a single missing square into four smaller boards that each now have exactly one missing square, which makes the subproblem identical to the original one.

## Worked Example

For `n = 2` with missing cell `(0, 0)`, the only valid tiling is `[[-1, 1], [1, 1]]`. For `n = 4` with missing cell `(0, 1)`, the recursion first places one center tromino, then solves the four `2 x 2` quadrants independently.

function tilingProblemLShaped(size, missingCell) {
  if (size <= 0 || (size & (size - 1)) !== 0) {
    throw new RangeError("Board size must be a positive power of two");
  }

  const [missingRow, missingColumn] = missingCell;

  if (
    missingRow < 0 || missingRow >= size
    || missingColumn < 0 || missingColumn >= size
  ) {
    throw new RangeError("Missing cell must lie inside the board");
  }

  const board = Array.from({ length: size }, () => Array(size).fill(0));
  board[missingRow][missingColumn] = -1;
  let tileId = 0;

  function solve(top, left, currentSize, holeRow, holeColumn) {
    if (currentSize === 2) {
      tileId += 1;

      for (let row = top; row < top + 2; row += 1) {
        for (let column = left; column < left + 2; column += 1) {
          if (row === holeRow && column === holeColumn) {
            continue;
          }

          board[row][column] = tileId;
        }
      }

      return;
    }

    const half = currentSize / 2;
    const midRow = top + half;
    const midColumn = left + half;
    const centers = [
      [midRow - 1, midColumn - 1],
      [midRow - 1, midColumn],
      [midRow, midColumn - 1],
      [midRow, midColumn],
    ];

    const holeQuadrant =
      holeRow < midRow
        ? (holeColumn < midColumn ? 0 : 1)
        : (holeColumn < midColumn ? 2 : 3);

    tileId += 1;
    const currentTile = tileId;

    for (let quadrant = 0; quadrant < centers.length; quadrant += 1) {
      if (quadrant === holeQuadrant) {
        continue;
      }

      const [row, column] = centers[quadrant];
      board[row][column] = currentTile;
    }

    const quadrantHoles = [
      holeQuadrant === 0 ? [holeRow, holeColumn] : centers[0],
      holeQuadrant === 1 ? [holeRow, holeColumn] : centers[1],
      holeQuadrant === 2 ? [holeRow, holeColumn] : centers[2],
      holeQuadrant === 3 ? [holeRow, holeColumn] : centers[3],
    ];

    solve(top, left, half, quadrantHoles[0][0], quadrantHoles[0][1]);
    solve(top, midColumn, half, quadrantHoles[1][0], quadrantHoles[1][1]);
    solve(midRow, left, half, quadrantHoles[2][0], quadrantHoles[2][1]);
    solve(midRow, midColumn, half, quadrantHoles[3][0], quadrantHoles[3][1]);
  }

  solve(0, 0, size, missingRow, missingColumn);
  return board;
}

## Follow-up Questions

- Why does one center tromino create exactly the right subproblem shape in all four quadrants?
- How would you prove correctness by induction on the board size?
- What changes if the board size is not a power of two?

## Common Pitfalls

- Marking the center tromino after the recursive calls instead of before them.
- Misidentifying which quadrant contains the original missing cell.
- Overwriting the designated missing square in the `2 x 2` base case.
