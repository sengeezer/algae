## Why It Surfaces

This problem tests whether you can define a DP over intervals instead of prefixes. For any substring, either the two ends match and contribute `2`, or one end must be excluded.

## JavaScript

```js
function longestPalindromicSubsequence(text) {
  const size = text.length;
  const dp = Array.from({ length: size }, () => new Array(size).fill(0));

  for (let index = size - 1; index >= 0; index -= 1) {
    dp[index][index] = 1;

    for (let end = index + 1; end < size; end += 1) {
      if (text[index] === text[end]) {
        dp[index][end] = dp[index + 1][end - 1] + 2;
      } else {
        dp[index][end] = Math.max(dp[index + 1][end], dp[index][end - 1]);
      }
    }
  }

  return dp[0][size - 1];
}
```

## TypeScript

```ts
export function longestPalindromicSubsequence(text: string): number {
  const size = text.length;
  const dp = Array.from({ length: size }, () => new Array<number>(size).fill(0));

  for (let index = size - 1; index >= 0; index -= 1) {
    dp[index][index] = 1;

    for (let end = index + 1; end < size; end += 1) {
      if (text[index] === text[end]) {
        dp[index][end] = end - index === 1 ? 2 : dp[index + 1][end - 1] + 2;
      } else {
        dp[index][end] = Math.max(dp[index + 1][end], dp[index][end - 1]);
      }
    }
  }

  return dp[0][size - 1];
}
```

## Interview Notes

- Define the state as the best answer inside `text[left..right]`.
- Call out the difference between this subsequence problem and longest palindromic substring.
- If asked for reconstruction, backtrack from the table edges inward.
