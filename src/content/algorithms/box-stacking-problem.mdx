## Why It Surfaces

Box stacking is a classic example of turning geometry into sequence DP. Once every box rotation is normalized into a base width and depth, the tallest stack is just the best predecessor chain where each box fits strictly inside the one below it.

## Worked Example

For one box with dimensions `(4, 6, 7)` and another with `(1, 2, 3)`, the smaller box can sit on top of a rotation of the larger one, so combining compatible rotations increases the total height.

```js
function boxStackingProblem(boxes) {
  const rotations = [];

  for (const [height, width, depth] of boxes) {
    const options = [
      [height, Math.max(width, depth), Math.min(width, depth)],
      [width, Math.max(height, depth), Math.min(height, depth)],
      [depth, Math.max(height, width), Math.min(height, width)],
    ];
    rotations.push(...options);
  }

  rotations.sort((left, right) => right[1] * right[2] - left[1] * left[2]);
  const bestHeight = rotations.map((rotation) => rotation[0]);

  for (let current = 0; current < rotations.length; current += 1) {
    for (let previous = 0; previous < current; previous += 1) {
      if (
        rotations[current][1] < rotations[previous][1] &&
        rotations[current][2] < rotations[previous][2]
      ) {
        bestHeight[current] = Math.max(
          bestHeight[current],
          bestHeight[previous] + rotations[current][0],
        );
      }
    }
  }

  return bestHeight.length === 0 ? 0 : Math.max(...bestHeight);
}
```

## Common Pitfalls

- Forgetting to generate all three box rotations.
- Comparing only one base dimension instead of both width and depth.
- Sorting by raw height instead of a stable base-ordering key before the DP.
