## Why It Surfaces

Euler's method is the entry-level numerical ODE solver. It is small enough for interviews, but still checks whether you can turn a recurrence formula into a careful step-by-step approximation.

## Worked Example

If `y' = x + y`, `y(0) = 1`, and the step size is `0.5`, the first Euler step is `1 + 0.5 * (0 + 1) = 1.5`.

```js
function eulerStep(xCurrent, stepSize, yCurrent, differentialEquation) {
  if (![xCurrent, stepSize, yCurrent].every((value) => typeof value === 'number' && Number.isFinite(value))) {
    throw new TypeError('State values and step size must be finite numbers');
  }

  if (stepSize <= 0) {
    throw new RangeError('Step size must be positive');
  }

  if (typeof differentialEquation !== 'function') {
    throw new TypeError('Expected a differential equation callback');
  }

  return yCurrent + stepSize * differentialEquation(xCurrent, yCurrent);
}

function eulerFull(xStart, xEnd, stepSize, yStart, differentialEquation) {
  if (![xStart, xEnd, stepSize, yStart].every((value) => typeof value === 'number' && Number.isFinite(value))) {
    throw new TypeError('Bounds, step size, and initial value must be finite numbers');
  }

  if (xEnd < xStart) {
    throw new RangeError('End x must be greater than or equal to start x');
  }

  const points = [{ x: xStart, y: yStart }];
  let xCurrent = xStart;
  let yCurrent = yStart;

  while (xCurrent < xEnd) {
    yCurrent = eulerStep(xCurrent, stepSize, yCurrent, differentialEquation);
    xCurrent += stepSize;
    points.push({ x: xCurrent, y: yCurrent });
  }

  return points;
}
```

## Follow-up Questions

- Explain how smaller step sizes trade runtime for accuracy.
- Compare Euler's method with higher-order schemes such as Runge-Kutta.
- Stop once `xCurrent + stepSize` would overshoot the target and handle the final fractional step separately.

## Common Pitfalls

- Using a zero or negative step size.
- Forgetting to keep both `x` and `y` in sync during each step.
- Treating Euler's method as exact instead of an approximation whose error depends on the step size.
