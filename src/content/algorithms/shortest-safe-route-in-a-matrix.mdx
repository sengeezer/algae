## Why It Surfaces

The trap here is that the obstacle set is not just the landmines themselves. Every neighbor of a landmine is unsafe too. Once you preprocess that rule, the remaining problem becomes a clean shortest-path search from multiple first-column starting points.

## Worked Example

On an all-ones `3 x 3` grid, every cell is safe, so the shortest safe route from the first column to the last column is `2` moves. For example, starting at `(0, 0)` and moving right twice reaches the last column immediately.

```js
function shortestSafeRouteInAMatrix(mat) {
  const rowCount = mat.length;

  if (rowCount === 0) {
    return -1;
  }

  const columnCount = mat[0].length;
  const directions = [
    [-1, 0],
    [1, 0],
    [0, -1],
    [0, 1],
  ];
  const safe = mat.map((row) => row.map((cell) => cell === 1));

  for (let row = 0; row < rowCount; row += 1) {
    for (let column = 0; column < columnCount; column += 1) {
      if (mat[row][column] !== 0) {
        continue;
      }

      safe[row][column] = false;

      for (const [rowDelta, columnDelta] of directions) {
        const nextRow = row + rowDelta;
        const nextColumn = column + columnDelta;

        if (nextRow >= 0 && nextRow < rowCount && nextColumn >= 0 && nextColumn < columnCount) {
          safe[nextRow][nextColumn] = false;
        }
      }
    }
  }

  const queue = [];
  const visited = Array.from({ length: rowCount }, () => Array(columnCount).fill(false));

  for (let row = 0; row < rowCount; row += 1) {
    if (!safe[row][0]) {
      continue;
    }

    queue.push([row, 0, 0]);
    visited[row][0] = true;
  }

  for (let head = 0; head < queue.length; head += 1) {
    const [row, column, distance] = queue[head];

    if (column === columnCount - 1) {
      return distance;
    }

    for (const [rowDelta, columnDelta] of directions) {
      const nextRow = row + rowDelta;
      const nextColumn = column + columnDelta;

      if (
        nextRow < 0
        || nextRow >= rowCount
        || nextColumn < 0
        || nextColumn >= columnCount
        || !safe[nextRow][nextColumn]
        || visited[nextRow][nextColumn]
      ) {
        continue;
      }

      visited[nextRow][nextColumn] = true;
      queue.push([nextRow, nextColumn, distance + 1]);
    }
  }

  return -1;
}
```

## Common Pitfalls

- Treating only the landmine cells as blocked and forgetting their neighbors.
- Starting BFS from every first-column cell instead of only the safe ones.
- Using DFS for a shortest-path problem with uniform edge costs.
