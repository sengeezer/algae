## Why It Surfaces

Palindrome Partitioning is a good interview problem because it combines two small ideas. You must choose where to cut the string, and every chosen piece must pass a palindrome test before the branch can continue.

## Worked Example

For `"aab"`, the valid partitions are `["a", "a", "b"]` and `["aa", "b"]`. The branch `["a", "ab"]` dies immediately because `"ab"` is not a palindrome.

## Reference Implementations

<ExamplePair>

```js
function palindromePartitions(text) {
  const results = [];
  const path = [];

  function isPalindrome(left, right) {
    while (left < right) {
      if (text[left] !== text[right]) {
        return false;
      }

      left += 1;
      right -= 1;
    }

    return true;
  }

  function dfs(start) {
    if (start === text.length) {
      results.push([...path]);
      return;
    }

    for (let end = start; end < text.length; end += 1) {
      if (!isPalindrome(start, end)) {
        continue;
      }

      path.push(text.slice(start, end + 1));
      dfs(end + 1);
      path.pop();
    }
  }

  dfs(0);
  return results;
}
```

```ts
export function palindromePartitions(text: string): string[][] {
  const results: string[][] = [];
  const path: string[] = [];

  function isPalindrome(left: number, right: number): boolean {
    let start = left;
    let end = right;

    while (start < end) {
      if (text[start] !== text[end]) {
        return false;
      }

      start += 1;
      end -= 1;
    }

    return true;
  }

  function dfs(start: number): void {
    if (start === text.length) {
      results.push([...path]);
      return;
    }

    for (let end = start; end < text.length; end += 1) {
      if (!isPalindrome(start, end)) {
        continue;
      }

      path.push(text.slice(start, end + 1));
      dfs(end + 1);
      path.pop();
    }
  }

  dfs(0);
  return results;
}
```

## Follow-up Questions

- Precompute palindrome substrings to reduce repeated checks.
- Return the minimum number of cuts instead of every partition.
- Explain how this differs from palindrome subsequence problems.

## Common Pitfalls

- Reusing a non-palindromic segment because the cut logic is correct.
- Forgetting to undo the last chosen segment on backtrack.
- Confusing substring partitions with subsequences.
