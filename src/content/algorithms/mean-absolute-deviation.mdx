## Why It Surfaces

Mean absolute deviation is a small statistics helper that tests whether you separate the center computation from the spread computation and whether you choose absolute differences rather than squared ones.

## Worked Example

For `[2, 34, 5, 0, -2]`, the mean is `7.8`. The absolute deviations from the mean average to `10.48`.

## Reference Implementations

<ExamplePair>

```js
function meanAbsoluteDeviation(values) {
  if (!Array.isArray(values) || values.length === 0) {
    throw new TypeError('Expected a non-empty array');
  }

  if (!values.every((value) => typeof value === 'number' && Number.isFinite(value))) {
    throw new TypeError('Array must contain only finite numbers');
  }

  const mean = values.reduce((total, value) => total + value, 0) / values.length;
  const absoluteSum = values.reduce((total, value) => total + Math.abs(value - mean), 0);
  return absoluteSum / values.length;
}
```

## Follow-up Questions

- Compare mean absolute deviation with variance and standard deviation.
- Return both the mean and the mean absolute deviation together.
- Explain why absolute deviations are sometimes easier to interpret than squared deviations.

## Common Pitfalls

- Forgetting to compute the mean before the deviations.
- Using squared differences instead of absolute differences.
- Allowing empty arrays and dividing by zero.
