## Why It Surfaces

Caesar cipher is the simplest wraparound string transformation. The real implementation detail is normalizing the shift and preserving non-letters instead of trying to rotate everything indiscriminately.

## Worked Example

With rotation `3`, `"Algae"` becomes `"Dojdh"`: each letter moves three positions forward, wrapping around from `Z` to `C` when needed.

## Reference Implementations

<ExamplePair>

```js
function caesarCipher(value, shift) {
  if (typeof value !== 'string' || !Number.isInteger(shift)) {
    throw new TypeError('Value must be a string and shift must be an integer');
  }

  const normalized = ((shift % 26) + 26) % 26;

  return value.replace(/[a-z]/gi, (char) => {
    const base = /[A-Z]/.test(char) ? 65 : 97;
    const index = char.charCodeAt(0) - base;
    return String.fromCharCode(base + ((index + normalized) % 26));
  });
}
```

## Follow-up Questions

## Common Pitfalls

- Failing to normalize shifts larger than 26 or negative shifts.
- Rotating punctuation and spaces that should stay unchanged.
- Rebuilding the alphabet map on every character instead of once per call.
