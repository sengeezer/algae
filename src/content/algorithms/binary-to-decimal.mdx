## Why It Surfaces

Binary-to-decimal conversion is the cleanest first example of positional notation. Every step says the same thing: shift the current value one place to the left in base 2, then add the next bit.

## Worked Example

Reading `1011` left to right gives `(((0 * 2 + 1) * 2 + 0) * 2 + 1) * 2 + 1 = 11`. That fold is Horner's method specialized for base 2.

## Reference Implementations

<ExamplePair>

```js
function binaryToDecimal(binaryString) {
  if (!/^[01]+$/.test(binaryString)) {
    throw new TypeError('Binary strings may only contain 0 and 1');
  }

  let value = 0;

  for (const digit of binaryString) {
    value = value * 2 + Number(digit);
  }

  return value;
}
```

```ts
export function binaryToDecimal(binaryString: string): number {
  if (!/^[01]+$/.test(binaryString)) {
    throw new TypeError('Binary strings may only contain 0 and 1');
  }

  let value = 0;

  for (const digit of binaryString) {
    value = value * 2 + Number(digit);
  }

  return value;
}
```

## Common Pitfalls

- Accepting invalid digits like `2` or whitespace.
- Recomputing powers of two unnecessarily for every position.
- Forgetting that the input is a string, not a base-10 number containing only zeros and ones.
