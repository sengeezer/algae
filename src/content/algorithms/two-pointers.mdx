## Why It Surfaces

Two-pointer problems reward you for preserving a simple invariant while avoiding nested loops. It is a natural fit for sorted arrays, mirrored strings, and linked-list scans where fast and slow pointers expose structure.

## JavaScript

```js
function pairSumSorted(nums, target) {
  let left = 0;
  let right = nums.length - 1;

  while (left < right) {
    const sum = nums[left] + nums[right];

    if (sum === target) {
      return [nums[left], nums[right]];
    }

    if (sum < target) {
      left += 1;
    } else {
      right -= 1;
    }
  }

  return null;
}
```

## TypeScript

```ts
export function isPalindrome(text: string): boolean {
  let left = 0;
  let right = text.length - 1;

  while (left < right) {
    if (text[left] !== text[right]) {
      return false;
    }

    left += 1;
    right -= 1;
  }

  return true;
}
```

## What To Say In Interviews

- State the invariant before you move either pointer.
- Explain why sorting or mirrored structure makes the two-pointer scan valid.
- Mention when a hash map would be better on unsorted data.
